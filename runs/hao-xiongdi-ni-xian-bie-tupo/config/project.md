# 《好兄弟，你先别突破》项目资料

## 项目定位

- 类型：双男主、好兄弟双穿、修仙升级、群像冒险、情境喜剧。
- 双男主关系：非 CP，十五年发小兼死党；对外一致，内部互损。
- 目标篇幅：约 500～520 章，120 万～150 万字。
- 喜剧重心：身份错位、默契闯祸、突破代症串台、随机感知串联、互相阻止升级。
- 长期结构：前期正常宗门与秘境成长，中期势力和群像扩张，后期才接触真正天劫与近乎失传的引劫传承。

## 一句话故事承诺

沈砚与谢停云同时魂穿修仙界，一个成为外门善后弟子，一个成为宗门天才，却被同命印绑定：一人突破，负面症状可能由另一人承担；同命印失稳时，两人的感知还会随机串联。两人从互相阻止突破，到学会把破境代价转给存在真实因果的敌人，最终发现无咎道祖一直把自己的高阶天劫转嫁给其他世界。

## 第一卷开篇路线（已确认）

- 第1章只用少量篇幅写两人在现代世界打完最后一局游戏，作用是建立十五年好友的互损、默契和私人暗号；随后两人分别以山地救援队员和现场安全评估、事故取证人员身份赶往同一处山体事故，在二次坍塌中互救死亡。现代游戏内容不得扩展为长期网络梗。
- 第2～3章分别完成沈砚在善后房、谢停云在首席洞府醒来的双穿身份建立。
- 第4～7章让沈砚自救、谢停云拖延筑基，因首席身份无法公开示弱而导致筑基阵试压提前。
- 第8章首次代症正式触发，谢停云的道基反噬转到沈砚身上，使沈砚再次濒死；重伤发生后降低喜剧密度，不以吐槽取消后果。
- 第9章谢停云只能根据异常反噬落在善后房的线索追到现场，仍查不到承受者姓名，因为沈砚在制度上尚属死者。
- 第10章两人以现代最后一局游戏的私人争执和坍塌前遗言进行双重验证，完成相认。
- 第11～13章双男主统一口供并公开装作不熟，陈百岁因拒绝改写沈砚伤情记录正式进入主线。
- 第14～15章宁知微因筑基阵出现无法解释的反噬缺口进入调查；裴问礼推迟到第16章以后正式登场，避免前十五章连续点名群像角色。
- 第一卷中沈砚必须拥有可见修炼增长：第1单元稳定练气二层循环，第2单元修复第一条主经脉并进入练气三层，第3单元完成第二条主经脉循环并进入练气四层；增长来自疗伤、修炼和资源结算，不来自无代价复制谢停云修为。

### 开篇笑点构建原则

- 小笑点来自双男主说话和行动习惯，大笑点来自计划、身份与宗门规则形成的行动连锁，长期笑点来自首席与名册死者必须公开装作不熟。
- 使用“认真目标＋身份限制＋合理误读＋行动升级＋真实后果”构建场景；删掉笑点后若主线因果完全不变，则该笑点必须重做。
- 同命印每次使用必须轮换感知类型、公开压力、承担角色和实际后果，不得固定为沈砚疼、谢停云看戏。
- “好兄弟，你先别突破”与现代“你先别冲”属于行动回调，每次出现必须改变突破准备、资源安排、战斗计划或关系。
- 陈百岁、宁知微和裴问礼不负责围观吐槽：陈百岁保住记录，宁知微识别阵法检测缺口，裴问礼拒绝利用丑闻或伤势夺位并承担失去机会的代价。

## 双男主

### 沈砚

- 现代身份：工程事故调查员。
- 魂穿身份：青崖宗善后房外门弟子。
- 开局境界：练气二层。
- 角色功能：计划、调查、资源管理、因果标记和转移路线封锁。
- 性格：谨慎、嘴损、擅长发现漏洞；看似稳重，大部分馊主意由他提出。
- 核心缺陷：害怕失控，习惯独自控制全部风险。

### 谢停云

- 现代身份：山地救援队成员。
- 魂穿身份：青崖宗首席天才。
- 开局境界：练气九层，正在准备筑基。
- 角色功能：正面战斗、危险行动、承受破境代价和炼化劫痕。
- 性格：胆大、手快、好面子；人前维持高冷天才形象，私下迅速恢复本性。
- 核心缺陷：习惯先救人再考虑代价，容易把冒险当成自己的义务。

## 同命印

- 同命印包含两层表现：低频但后果重的“代症”，以及高频但不直接改写修为的“串感”。
- 代症是主规则：前期只会在两位男主之间自动转移重大破境产生的负面症状。
- 低中境界转移的是经脉反噬、灵力失控、丹火、丹毒、神魂撕裂、心魔和法则反噬，不称为天劫。
- 串感是日常喜剧规则：同命印不稳定时，可随机共享痛觉、冷热、味觉、嗅觉、视听以及未经处理的神识信息。
- 串感不等于读心。双方不能自动听见彼此想法；记忆只允许在重大心魔事件中有限接通。
- 低阶修士接到高阶神识时，只会承受远超自身处理能力的原始信息，不会因此永久获得高阶神识范围、判断速度或修为。
- 一方死亡不会让另一方立即死亡，但会使其道基永久残缺。
- 恶意连续突破会触发双向加倍反噬。
- 中期可以把破境代价转向已建立真实因果的敌人。
- 转移不消除代价；连续转移会提高强度，失败时可能同时反噬双方。
- 同命印是双男主魂穿时，两道神魂共同撕下一块高阶劫道权柄后形成的异常，不属于常规修炼体系。

### 同命印成长路线

1. 被动代症：重大破境的负面症状在双男主之间自动转移。
2. 随机串感：同命印失稳时，痛觉、五感或神识原始输入随机接通。
3. 局部控制：两人逐步学会主动开关部分感知，但仍不能借此读心或永久复制能力。
4. 因果转症：中期可把负面症状、负面感知转给存在真实因果的敌人。
5. 引劫合流：后期结合极少数引劫人的传承，处理合道之后出现的真正天劫。

## 天劫边界

- 筑基、金丹、元婴等低中境界没有真正天劫。
- 合道阶段才首次接触天地降下的真正天劫。
- 渡劫阶段面对完整天地检验。
- 前中期不得用雷云、灭世天象或高阶天劫代替正常破境过程。

## 引劫一脉

- 引劫是一门近乎失传的高阶旁门，不是公开职业，也没有学院、行业或普及体系。
- 下界已知仍存活的正式引劫人只有三位，全书正式登场的传承者不超过七人。
- 引劫人至少达到化神后，才可能真正接触并牵引天劫。
- 普通引劫人必须提前知道渡劫时间，使用大型阵法、承劫法器和固定落点，只能移动部分天劫。
- 双男主后期以同命印作为活体中转节点，结合引劫传承，才能即时改变完整天劫的目标。

### 已确认的重要引劫人

- 周既明：隐居青崖宗的落魄长老，只能感知和轻微牵引劫机。
- 桑婆：能够把少量天劫储入法器，每次使用都会造成永久伤损。
- 闻人渡：无咎道祖下属，能够通过多年阵法准备移动高阶天劫。

## 群像与阶段角色管理

- 全书结构正式确定为“双男主为轴、阶段群像轮换”，不是纯双男主冒险，也不是固定小队从头跟到尾。
- 只有沈砚、谢停云是全书持续在线的固定核心。
- 其他角色按分卷或故事单元进入主线，不要求长期跟队，也不得为了维持存在感无目的伴随双男主。
- 每名阶段角色在分卷细纲中必须记录：首次登场、活跃区间、独立目标、退出原因与状态、回归条件、信息上限。
- 角色退出后允许持续影响剧情，但不必持续出场；回归必须由自己的目标、旧因果或新证据触发。
- 群像必须贯穿前期、中期、中后期和终局，每卷安排二至四名阶段核心角色，不能只在双男主分线阶段集中出现。
- 每名阶段核心角色至少拥有一次独立选择、一次由自身承担的代价和一次可见兑现；删除该角色后若主线事件仍可原样发生，则该角色不算有效群像。
- 每卷至少有一个关键结果由阶段角色主动促成，双男主可以补救、放大或承担后果，但不能包办所有解法。
- 阶段角色允许与双男主意见冲突、拒绝合作或选择离队，不得永远充当听令执行者。
- 终局回归角色必须来自前文已完成独立弧线的人物；最后一卷不得临时增加一批只负责助战的新核心角色。

### 已确认的阶段角色

- 宁知微：前期宗门单元登场，负责突破安全与阵法改革线；完成阶段方案后回到家族或阵法改革现场，只有当阵法证据、突破安全或家族线再次进入主线时回归。她能研究同命印的现象，不能独立解释其高阶来源。
- 罗大有：前中期受害者单元登场，提供被强者转嫁修炼反噬的亲历视角；脱离控制后转入保护同类受害者的行动线，只有受害者网络或幕后因果链推进时回归。他知道转嫁造成的现实伤害，不掌握引劫传承的全貌。
- 七号：中后期只先留下编号与造物痕迹，合道阶段以后才正式登场；完成自我意识与来源单元后可离队独立选择去向，只有无咎道祖的天劫网络或劫力证据需要直接见证者时回归。
- 周既明：前期只在一个宗门单元内提供第一层异常线索，随后因旧伤反噬、主动断线或隐藏调查而退出；中后期证据链补齐时才允许回归。他知道有人篡改劫道、知道引劫一脉残存，但不知道无咎道祖的完整身份、全部转移网络和最终计划。

### 分卷补全的阶段角色

- 陈百岁（`chen-baisui`）：第1～2卷活跃，善后房旧主事，目标是让死者留下真实姓名与死因；拒绝销毁沈砚记录后被撤职，转为游历收殓人。第4卷因镇脉峰死者名单回归并完成作证，之后退出。他只掌握青崖宗事故记录，不知道高阶转移网络。
- 裴问礼（`pei-wenli`）：第1卷以首席竞争者登场，第3卷秘境回归，第6卷边境线完成弧线；目标是摆脱谢停云陪衬身份并通过公平竞争证明自己的剑道。多次因先救人后争名失去排名，最终抗命保住部下并失去宗门身份。他不知道魂穿真相和引劫传承。
- 顾采薪（`gu-caixin`）：第2～3卷活跃，第5卷因承厄楼丹毒线回归；目标是摆脱债务丹房并成为独立丹师。公开解毒方后失去商号利益，取得个人丹方与名声，之后留在仙城行医而不长期跟队。她只掌握药物、丹毒和普通转症伤害。
- 孟观澜（`meng-guanlan`）：第3卷秘境守护者，第7卷因失落界门与秘境相连而回归；目标是保护秘境居民和传承不被宗门抽空。开放祖传禁区后失去部分继承权，最终放弃控制权换取秘境存续。他只知道古代阵纹与界门碎片。
- 陶朱衣（`tao-zhuyi`）：第5～6卷活跃的仙城商人，目标是夺取承厄楼合法产业而非无私改革。她帮助取得账册、建立普通资源渠道并带走皇朝部分生意，收益兑现后退出；只有物资封锁或旧契约再次进入主线时回归。她不知道引劫与无咎道祖身份。
- 赵明夷（`zhao-mingyi`）：第6卷皇朝档案官，目标是证明皇朝法令不能把弱者登记为合法耗材。公开代灾总册后流亡，完成个人选择后退出；后续只通过公开档案和政治后果影响剧情。他不知道跨界网络全貌。
- 唐回山（`tang-huishan`）：第7卷远征队领队，目标是把失踪队员带回去而非争夺遗迹。放弃高阶遗物后被撤职，但保存整支队伍；除非出现与原远征队直接相关的跨界救援，否则不回归。他不知道无咎道祖与原世界坐标。

### 揭秘接力

- 周既明：交付“破境代价可以被改道”的第一层线索。
- 宁知微：从阵法与灵力残留证明改道现象能够被人为利用。
- 罗大有：补足受害者链与现实后果，证明这不是孤例。
- 桑婆：补足引劫一脉的历史、代价与能力边界。
- 七号：提供天劫被制造、储存和跨界投送的直接证据。
- 闻人渡：把证据链连接到无咎道祖及其转移网络。
- 任何单一角色都不得从头到尾负责解密；双男主必须通过调查、误判、验证与代价逐层拼出真相。

## 长篇阶段

### 前期（约 1～180 章）

- 双穿身份适应、宗门生存、外门与内门竞争、秘境和资源争夺。
- 两位男主互相承担普通破境反噬，并试图阻止对方升级。
- 不出现真正天劫，不公开引劫流派。

### 中期（约 181～320 章）

- 金丹、元婴成长，进入仙城、皇朝和宗门战争。
- 同命印开始把普通破境代价转给有真实因果的敌人。
- 周既明只以传说和残留证据形式引出引劫人。

### 中后期（约 321～420 章）

- 双男主达到化神、炼虚，开始接触天地法则。
- 桑婆与闻人渡登场，引劫一脉成为重要但仍小众的秘密线。
- 发现无咎道祖一直在清除能够追踪天劫去向的人。

### 后期（约 421～520 章）

- 双男主达到合道，第一次面对真正天劫。
- 同命印与引劫传承合流，形成独有的即时天劫中转能力。
- 无咎道祖准备把最终天劫转移给双男主原来的世界。
- 双男主将无咎道祖历代逃避的天劫全部送回，并在正面战斗中击败其万劫魔身。

## 终局方向

- 无咎道祖是最终 Boss，也是一直秘密消灭引劫传承的人。
- 他每次高阶突破产生的天劫都被转移到其他世界。
- 双男主原来的世界已被标记为下一处承劫地，他们的魂穿来自转移网络发生的异常。
- 终局不是废除同命印，而是摧毁能够向无辜世界大规模转移天劫的网络。
- 两位男主保留自己的转移能力，只允许把代价送回真正制造或受益的因果主体。

## 原世界与魂穿闭环

- 双男主穿越前死于同一场山体二次坍塌：谢停云参加现场救援，沈砚因事故调查工作留在现场；坍塌同时受到无咎道祖跨界网络坐标试投放的干扰。
- 两人的现代身体已经死亡，终局所谓返回原世界，是以修仙界身体通过安全跨界通道进入，不是重新回到原身体。
- 两界平均时间比例约为十比一但并非绝对恒定；全书在修仙界约经历一百至一百二十年，原世界约过去十至十二年。
- 原世界灵气稀薄、没有高阶修士感知劫力，同时拥有密集生命因果，因此被无咎道祖选为承劫地；天劫会表现为全球灾害、法则崩坏和生命衰竭，不是让普通人直接获得或吸收修士雷劫。
- 两人互相救援时共同扯下一块跨界转移权柄，权柄进入青崖宗附近后自动寻找与改道痕迹接触最深且神魂刚刚消散的兼容身体，由此形成同命印。
- 沈砚原身因处理代伤阵尸体而被灭口；谢停云原身曾在秘境接触另一块改道残片，被师门抹去部分记忆，并在强行筑基前神魂崩溃。
- 无咎道祖没有主动挑选或培养双男主，直到中后期才确认权柄碎片落在他们身上。
- 第二、四、六、八卷依次出现临终救援声音、现代坐标图形、无灵界记录和原世界直接影像，逐步建立终局故乡线。

## 双男主缺陷递进

- 沈砚：第二卷的问题是沟通不足；第四卷升级为替伙伴决定风险；第五卷升级为自认能够客观定义因果，却忽略受害者的选择权，不能连续用“隐瞒计划”重复表现控制欲。
- 谢停云：第三卷仍把自己当诱饵；第四卷尝试独自承担危险仍然失败；第六卷开始通过分工、撤离和接受战略损失救人；第九卷承担真正天劫必须建立在明确授权和共同计划之上。

## 关键结丹代症
