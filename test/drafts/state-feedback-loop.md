# state-feedback-loop

## 测试内容

章节状态回填联动测试：验证正文写完后，状态记录能否反哺下一章。

## 正文样例

王兆陪从梁存义旧斋出来时，天已经黑了。

怀里那张旧渠图不重。

压在身上却比一袋粮还沉。

周福在巷口等他，见他出来，赶紧迎上来：“先生跟你说什么了？”

王兆陪没有立刻答。

他回头看了一眼旧斋的灯。

“他说，赵管事不是根。”

周福脸色微变。

这句话，他听懂了一半。

另一半，他不敢懂。

“那你还查吗？”

王兆陪把旧渠图往怀里按了按。

“查。”

“从哪查？”

“从水查。”

周福愣住：“不是查租子？”

“租子是刀，水才是柄。”王兆陪道，“谁握着水，谁就握着田。谁握着田，谁就能让小户一年一年把地吐出来。”

周福听得后背发凉。

他只是个书铺掌柜，最会算的是纸墨账。

可现在，他忽然发现，王兆陪要算的账，比书铺里那点账大太多了。

王兆陪停下脚步。

“周掌柜，俺也去要你帮我一件事。”

“你说。”

“明天开始，不要再传俺也去会写诗。”

周福一愣：“这可是好名声。”

“换个说法。”

“换成什么？”

王兆陪抬眼，看向巷口外的夜色。

“传俺也去在查青石旧渠。”

周福脸色一下白了。

诗名会引来读书人。

旧渠，会引来真正吃田的人。

“你这是要把火往自己身上引。”

“不引出来，怎么知道谁怕火？”

王兆陪说完，迈步往外走。

巷口风冷。

可他心里反倒稳了。

因为下一步，不再是被人推着走。

是他自己把线抛出去。

## 写后状态回填

- 人物状态变化：
  - 王兆陪从“解决王家租子危机”升级为“意识到地方旧网存在”
  - 周福从纯粹帮忙扬名，转为开始参与风险更高的信息传播
- 关系变化：
  - 王兆陪与梁存义建立初步信任
  - 王兆陪与周福的合作加深，但风险上升
- 资源变化：
  - 新增资源：青石旧渠图
  - 新增信息：赵家并田可能与水渠、旧账相关
- 新伏笔：
  - 谁控制旧渠
  - 赵家背后是否有更高层人物
- 已回收伏笔：
  - 为什么赵管事急着压王家
- 下一章前置状态：
  - 读者知道王兆陪准备主动放出“查旧渠”的风声
  - 赵家暂时还不知道王兆陪掌握旧渠图

## 下一章承接测试

下一章应从“风声放出后，赵家第一个异常反应”开始，而不是重新解释旧渠背景。
