# 项目资料：T12《灵田账房今天也没挨打》

## 测试定位

- 类型：修仙成长 + 情境喜剧 / 角色喜剧。
- 叙事：第三人称限知，以许观为主视角。
- 语言：轻快、克制、口语自然；笑点来自规则、行动和人物反差，不写网络热梗。
- 首个故事单元：10章，每章正文目标2000～2500非空白字符。
- 阶段承诺：许观必须在灵田枯苗和公物失窃的双重指控下保住杂役身份，并取得外门小考资格。

## 核心角色

### 许观（稳定ID：xu-guan）

十七岁，归元宗青禾谷杂役，炼气一层。贫穷、谨慎、算账极快，遇到荒谬场面时越紧张越一本正经。想成为外门弟子，不是为了威风，而是外门每月有固定修炼配额，能让他不再靠赊账修炼。优点是观察细、能承担后果；缺点是过度计算风险，常把一句普通提醒说成债务清算。

他会基础吐纳和残缺的“听脉术”：能短暂听见灵气在土壤、经脉或器物中的流向，但使用后会耳鸣、判断距离失真，严重时鼻血和短暂失聪。此术只能发现流动异常，不能直接知道凶手、动机或完整事实。

### 庞大海（稳定ID：pang-dahai）

许观同屋杂役，炼气一层。嘴快、爱吹嘘、怕疼，但真正危急时会留下帮忙。他把每次害怕都解释成“替大家保存有生力量”，说话夸张，行动常比嘴诚实。

### 宁霜（稳定ID：ning-shuang）

外门巡田弟子，炼气三层。说话简短、按证据办事，不替许观开脱。她对宗门规章熟悉，却厌恶有人拿规章掩盖责任。她的冷面反应可以制造反差，但不能写成不通人情或无条件帮助主角。

### 严九尺（稳定ID：yan-jiuchi）

青禾谷杂役管事，炼气二层。迷信表格、封签和口头流程，认为“记录存在就代表事情受控”。他不一定是幕后坏人，但会为了保住管事位置推卸责任。他说话喜欢引用条目编号，越心虚引用得越精确。

### 罗七（稳定ID：luo-qi）

灵材库跑腿杂役，炼气二层。办事圆滑，习惯利用信息差赚取小利。首单元中是主动对手之一，但他的具体责任必须由正文证据逐步确认，不能开局由旁白直接宣布。

## 开局事实

- 许观负责的七号灵田在外门巡查前夜大面积枯苗。
- 同一晚，公用库少了四张引雨符；借用登记最后一笔写着许观的工牌号。
- 许观没有拿符，但他曾为省贡献点，私下修补过七号田的旧引水竹管，因此确实违反了“公用灵田设施不得私修”的规定。
- 枯苗若在三日内无法止住，许观需要赔偿并被逐出青禾谷；外门小考推荐也会取消。
- 真相方向：引水管的灵压异常、错误批次的引雨符和有人倒卖合格符共同造成问题。任何单一证据都不能立刻解决全部指控。

## 首单元硬约束

- 必须有一次明确受挫：许观找到的初步物证失效或反噬，他受伤且贡献点被冻结，不能无代价翻盘。
- 必须有一次公开资格/证据核验，许观要在对手和规章压力下证明证据链，而非靠高层一句话保人。
- 第10章形成可见阶段收束：枯苗危机得到控制，盗符主责有阶段结论，许观取得外门小考资格。
- 收束必须保留代价：伤势、债务、被更高层注意或新限制至少保留两项。
- 许观在本单元不得突破到炼气二层；可以提升控制力或形成新理解，但不能跨境。
- 宁霜不能替许观完成调查；庞大海不能只负责插科打诨；严九尺和罗七不能写成只会自爆的蠢人。
- 严肃受伤、赔偿和逐出风险出现时，笑点密度必须下降，笑点不能取消后果。

## 长期方向（只作边界，不在首单元解决）

许观逐渐发现归元宗底层资源分配不是单纯的某个坏人造成，而是规则、短缺和逐利共同形成。他的成长方向是从“只算自己能不能活下去”，变成能看懂并改变更大范围的资源流动。喜剧长期来自他用账房思维理解修仙世界，以及不同角色对规则的不同执行方式。
