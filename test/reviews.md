# 测试评价

## ch01-v1

- 测试内容：原始倾向下的首章开局
- 主要问题：
  - 中段解释偏多
  - 农具卖点进入偏晚
  - 只有钩子，兑现不够硬
- 评价：不通过

## ch01-v2

- 测试内容：补入“更快冲突、更早卖点、更少说明”后的首章
- 改善点：
  - 催租堵门更早进入
  - 改犁卖点更早出现
  - 有了小兑现
- 剩余问题：
  - 仍有一点解释感
  - 钩子仍可更狠
- 评价：部分通过

## ch01-v3

- 测试内容：进一步加入“差异动作”和“更硬小兑现”的首章
- 改善点：
  - 主角直接拆犁
  - 不是只说会改，而是先证明一部分
  - 钩子明确指向下一章
- 剩余问题：
  - 家庭反应仍偏功能化
- 评价：通过

## ch01-v4

- 测试内容：按更新后 Prompt 正式写的验证首章
- 改善点：
  - 前段抓力最强
  - 危机、差异化动作、专业点、对手全部落位
  - 本章兑现与章末钩子都成立
- 剩余问题：
  - 技术解释仍要控制密度
- 评价：通过，可作为当前首章写法基线

## ch02-verify

- 测试内容：借工具、连夜改犁
- 结论：
  - 未掉回解释流
  - 专业感落在动作里
  - 有明确兑现点
- 评价：通过

## pressure-chapter

- 测试内容：赵管事夜里拆台
- 结论：
  - 施压具体
  - 有现场比试和结果兑现
  - 章末明确推下一步
- 评价：通过

## poem-breakout

- 测试内容：县城文会被逼上台，当场作诗
- 结论：
  - 没写成机械震惊流
  - 诗词服务身份翻盘
  - 本章拿到了圈层门缝
- 评价：通过

## multi-goal-chapter

- 测试内容：诗名外传 + 农具线出问题 + 人际关系推进 + 下一章钩子
- 结论：
  - 多目标没有互相打架
  - 不是单纯说明会
  - 节奏仍然稳定
- 评价：通过

## emotion-relationship

- 测试内容：轻情绪/关系推进章
- 结论：
  - 没有强冲突，但父子关系发生可感知变化
  - “先不烧书”和“站到犁后看”构成小兑现
  - 章末以全村围观试犁继续推进
- 评价：通过

## batch-3chapters

- 测试内容：连续 3 章组测
- 结论：
  - 三章没有每章重新开局
  - 每章都有独立目标和兑现点
  - 连续阅读节奏稳定
- 评价：通过

## failure-setback

- 测试内容：失败/受挫章
- 结论：
  - 主角当众失败，但没有崩盘
  - 失败被转化为问题定位
  - 章末保留“第二版下午复试”的明确钩子
- 评价：通过

## multi-line-merge

- 测试内容：多线正式交汇章
- 结论：
  - 诗名线、农具线、旧账线在同一事件中交汇
  - 贵人到场没有写成单纯救场
  - 多线未明显混乱
- 评价：通过

## court-politics

- 测试内容：朝堂/权谋试探章
- 结论：
  - 有具体议题：三府秋粮、亏空、运粮脚费
  - 有立场陷阱：查谁、谁去查、如何站队
  - 主角以制度性方案拆局，而非直接开骂
- 评价：通过

## long-info-reveal

- 测试内容：长信息揭示章
- 结论：
  - 信息通过县志、旧渠图、师生问答逐层释放
  - 揭示改变了主角目标认知
  - 没有写成大段背景说明
- 评价：通过

## state-feedback-loop

- 测试内容：状态回填联动测试
- 结论：
  - 人物、关系、资源、伏笔、下一章前置状态都有记录
  - 下一章可直接承接“放出查旧渠风声”
- 评价：通过

## motif-fatigue

- 测试内容：同一卖点连续使用的疲劳测试
- 结论：
  - 连续种地卖点必须切换兑现角度
  - 如果重复“看出问题-解释-众人震惊”会疲劳
  - 当前建议记录为后续 Prompt 可选增强项
- 评价：通过
