# 新模具复跑细纲 v1：第111~115章

> 用途：承接第110章“你是来写人，还是来写路”的门槛，验证局内后的第一次表态、现实动作和新阶段钩子是否还能连续成立。

## 批次目标

- 验证局内之后是否还能保持场景推进
- 验证“写人 / 写路”的选择是否能落到具体行动
- 验证更高层压力下正文是否不退成空谈

## 批次风险点

- 高层问话容易写成纯理念
- 主角表态容易写成口号
- 局内压力容易只剩解释，没有现实反咬

---

## 第111章：先写路
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：面对“写人还是写路”，王兆陪没有急着点名谁，而是先把路写出来
- **本章兑现点**：主角第一次在局内明确自己的落笔顺序
- **最晚冲突落点**：前30%
- **核心事件**：主角回答局内主位人物的问题，解释为什么先写路，再写人；对方要求他现场改一段口径
- **冲突类型**：局内表态 / 口径再压
- **人物发展**：主角从被问，转为主动定自己的写法
- **信息释放**：局内人真正看的不是他敢不敢咬人，而是能不能把路写得别人接得住
- **钩子**：主位人物让他把“人名”先划掉，只留“路数”
- **本章不可删场景**：主角现场回答“写人还是写路”
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角站在局内见面场，被问“写人还是写路”
  - 必做事件：
    1. 回答为什么先写路
    2. 被要求现场改口径
    3. 落下“划掉人名，只留路数”的新要求
  - 禁止越界：
    1. 不直接点出幕后最终主使
    2. 不直接完成局内接纳
  - 结束状态：主角进入局内第一轮实操
  - 下章承接：围绕划掉人名后的稿子进入现实反咬
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 问题压场，负责承接上章钩子
    2. 主角表态，负责立写法
    3. 现场改口径，负责把理念变成动作
  - 字数支撑理由：本章有问答、判断、实操三段，不靠空谈撑体量
  - 场景级字数预算：
    1. 场景1：约600~750字
    2. 场景2：约800~1000字
    3. 场景3：约500~700字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。现场改稿能提供动作量。

## 第112章：划掉人名以后
- **功能**：压
- **目标字数**：2000~2500字
- **本章即时卖点**：人名一划掉，稿子反而更危险，因为它开始指向一整条路
- **本章兑现点**：展示“写路”比“写人”更难压
- **最晚冲突落点**：前35%
- **核心事件**：主角按要求重整底稿，发现删掉人名后，证据反而能串成更清晰的仓路
- **冲突类型**：结构重写 / 证据重排
- **人物发展**：主角意识到真正有力的不是骂谁，而是让路数自己露出来
- **信息释放**：局内人给的不是退让，而是更稳的写法
- **钩子**：删改后的稿子刚成，外头传来鲁伙计可能被转入府仓旧院的消息
- **本章不可删场景**：删掉人名后的证据重排必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角被要求先划掉人名
  - 必做事件：
    1. 删除人名
    2. 重排证据线
    3. 用鲁伙计新消息把稿子拉回现实
  - 禁止越界：
    1. 不直接救回鲁伙计
    2. 不直接让局内主位人物出手
  - 结束状态：主角发现写路会牵出更深现实入口
  - 下章承接：围绕府仓旧院进入行动
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 划名与重排，负责结构动作
    2. 证据路数显形，负责兑现写路价值
    3. 旧院消息入门，负责现实反咬
  - 字数支撑理由：本章有删改、判断、消息三段推进
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。重排证据是明确剧情动作。

## 第113章：府仓旧院
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：写路刚有形，现实里的旧院就把这条路咬了回来
- **本章兑现点**：让“写路”第一次对应到一个真实地点
- **最晚冲突落点**：前25%
- **核心事件**：主角和邢管事沿消息探到府仓旧院，发现这里并非关人正处，而是转账、转物、转口供的中转点
- **冲突类型**：现实探点 / 线索落地
- **人物发展**：主角学会用稿子里的路数反查现实地点
- **信息释放**：府仓旧院是仓路改写链中的关键中转
- **钩子**：旧院墙缝里发现一枚与鲁伙计衣襟相同的线头
- **本章不可删场景**：旧院探点必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角拿到府仓旧院消息
  - 必做事件：
    1. 进入旧院外围探点
    2. 确认旧院不是普通空院
    3. 找到鲁伙计相关线头
  - 禁止越界：
    1. 不直接找到鲁伙计本人
    2. 不直接揭开所有账路
  - 结束状态：主角确认写路能反查现实
  - 下章承接：围绕旧院线头和中转用途继续追
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 旧院外围观察，负责拉现实感
    2. 入内探线，负责发现中转痕迹
    3. 线头入手，负责给下一章钩子
  - 字数支撑理由：本章是行动探点章，天然有场景动作
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。旧院场景足够支撑正文。

## 第114章：旧院不是终点
- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：旧院不是藏人的终点，而是把人和账都转走的中间口
- **本章兑现点**：把旧院线从地点推进到中转规则
- **最晚冲突落点**：前30%
- **核心事件**：主角通过旧院痕迹和账路推断，确认这里只负责转，不负责留；同时意识到下一步要找“谁有资格让它转”
- **冲突类型**：规则推断 / 中转追索
- **人物发展**：主角从找地点升级为找调度规则
- **信息释放**：仓路真正的关键不是某个院，而是谁能让这些院动起来
- **钩子**：局内人递回一句：“你终于开始写路了。”
- **本章不可删场景**：旧院中转规则的推断必须落在实物和痕迹上
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角拿到旧院线头
  - 必做事件：
    1. 推断旧院是中转口
    2. 把中转规则和底稿路数对上
    3. 收到局内人的反向确认
  - 禁止越界：
    1. 不直接找到调度者
    2. 不直接救人
  - 结束状态：主角确认写路方向正确
  - 下章承接：围绕谁有资格调动旧院进入新试探
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 旧院痕迹复查，负责建立事实
    2. 中转规则推断，负责把事实变成路数
    3. 局内确认，负责落下阶段反馈
  - 字数支撑理由：本章不是纯推理，依赖痕迹、账路和反馈三段
  - 场景级字数预算：
    1. 场景1：约600~750字
    2. 场景2：约850~1050字
    3. 场景3：约450~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。规则推断有实物支撑。

## 第115章：谁能让旧院动
- **功能**：破
- **目标字数**：2000~2500字
- **本章即时卖点**：找到了旧院还不够，真正要紧的是谁能让旧院动起来
- **本章兑现点**：把“写路”推进到“找调度者”的下一阶段
- **最晚冲突落点**：前35%
- **核心事件**：主角把旧院中转、残纸改账、鲁伙计失踪三线并到一处，确认下一步不是找某个人，而是找能调动旧院的权柄入口
- **冲突类型**：阶段收束 / 新目标建立
- **人物发展**：主角正式从写路进入查调度权
- **信息释放**：真正的局不在一个人名，而在能让多处节点一起动的权柄
- **钩子**：常先生终于让人递来第一句明话：“明日来见我。”
- **本章不可删场景**：三线并合必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角确认旧院是中转口
  - 必做事件：
    1. 合并旧院、残纸、鲁伙计三线
    2. 建立“调度权”这个新目标
    3. 章末收到常先生明面召见
  - 禁止越界：
    1. 不直接见常先生
    2. 不直接解决鲁伙计线
  - 结束状态：主角从写路推进到查调度权
  - 下章承接：常先生明面召见
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 三线复盘，负责收束信息
    2. 调度权目标建立，负责给下一阶段方向
    3. 常先生召见，负责抛出新阶段钩子
  - 字数支撑理由：本章是阶段收束章，但有三线合并和新目标建立，不是总结
  - 场景级字数预算：
    1. 场景1：约650~850字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。三线并合有足够信息量。

## 本批次备注
- **批次范围**：第111~115章
- **批次目标**：验证局内后的表态、写路实操、现实反咬和阶段收束能否连续成立
- **批次风险点**：第111章和第114章容易写成理念对谈，正文试写时要优先检查是否有现场动作
- **对下一批的约束**：如果“写路”仍然偏抽象，下批应优先把常先生明面召见写成实场景和实压力
