# 新模具复跑细纲 v1：第121~125章

> 用途：承接第120章“等对方动”，验证税栈暴露后，对方的转移、反扑、诱饵和鲁伙计现实危机是否能连续成立。

## 批次目标

- 验证对方不是被动等待，而是会主动转移和反扑
- 验证鲁伙计线继续保持现实压力
- 验证常先生减少解说后，局势本身能否推动正文

## 批次风险点

- 追索重复
- 对方反应不够强
- 鲁伙计继续只作为线索存在

---

## 第121章：他们动了
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪刚学会等，对方就真的动了
- **本章兑现点**：税栈线暴露后，对方开始转移可疑人和物
- **最晚冲突落点**：前25%
- **核心事件**：税栈夜里突然清货，主角和邢管事远远盯住，发现一辆轻车从后门走出
- **冲突类型**：对方转移 / 远距离盯梢
- **人物发展**：主角第一次用“等”逼出对方动作
- **信息释放**：税栈线已经触到对方痛处
- **钩子**：轻车没有往外城走，反而绕回府仓旧院方向
- **本章不可删场景**：夜里税栈清货和轻车转移必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角等待对方因税栈暴露而动
  - 必做事件：
    1. 税栈夜间清货
    2. 远距离盯住轻车
    3. 发现轻车回绕旧院
  - 禁止越界：
    1. 不直接截车
    2. 不直接救回鲁伙计
  - 结束状态：主角确认对方已经开始转移
  - 下章承接：围绕轻车回绕旧院继续追
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 税栈异动，负责兑现等待
    2. 远距盯梢，负责行动张力
    3. 轻车回绕，负责新钩子
  - 字数支撑理由：本章是对方主动动作章，行动链完整
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。盯梢和转移足够支撑正文。

## 第122章：旧院回车
- **功能**：争 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：车从税栈出来，却绕回旧院，说明这不是逃，而是洗
- **本章兑现点**：证明旧院和税栈之间存在循环转移
- **最晚冲突落点**：前30%
- **核心事件**：主角追到旧院外，发现轻车卸下的不是人，而是能伪装人走过的衣物和旧物
- **冲突类型**：诱饵识别 / 转移循环
- **人物发展**：主角开始识别对方的假转移
- **信息释放**：对方想把追索引回旧院，让主角误判鲁伙计方向
- **钩子**：旧物里有一只鞋，但鞋底泥色与税栈不合
- **本章不可删场景**：旧院回车和假物卸下必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角跟住轻车回旧院
  - 必做事件：
    1. 观察旧院回车
    2. 发现卸下的是诱饵旧物
    3. 用鞋底泥色识破误导
  - 禁止越界：
    1. 不直接确认鲁伙计所在
    2. 不直接抓住收尾人
  - 结束状态：主角确认对方在做假转移
  - 下章承接：围绕鞋底泥色反查真正方向
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 旧院外盯车，负责承接
    2. 卸物观察，负责识别诱饵
    3. 泥色判断，负责反推方向
  - 字数支撑理由：本章有观察、识破、反推三段
  - 场景级字数预算：
    1. 场景1：约600~750字
    2. 场景2：约800~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。诱饵识别有明确动作。

## 第123章：泥色不对
- **功能**：破
- **目标字数**：2000~2500字
- **本章即时卖点**：鞋是鲁伙计的，泥却不是税栈和旧院的泥
- **本章兑现点**：用细节反推出第三处真正落点
- **最晚冲突落点**：前35%
- **核心事件**：主角通过泥色、草籽和水痕判断，真正转移方向不是旧院，而是靠近西河渡的小堆场
- **冲突类型**：细节反推 / 第三地点建立
- **人物发展**：主角进一步从追迹者变成判迹者
- **信息释放**：对方不是单线转移，而是用旧院做假口，真口在西河渡
- **钩子**：西河渡当天夜里有一批无主空篓临时入账
- **本章不可删场景**：泥色判断必须落在具体细节上
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角拿到鞋底泥色不合的线索
  - 必做事件：
    1. 比对泥色
    2. 判断第三地点
    3. 接入西河渡空篓账
  - 禁止越界：
    1. 不直接找到鲁伙计
    2. 不直接揭开调度者身份
  - 结束状态：主角转向西河渡
  - 下章承接：进入西河渡堆场
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 泥色比对，负责立证据
    2. 第三地点判断，负责转向
    3. 空篓账接入，负责给新入口
  - 字数支撑理由：本章是细节推理章，但所有判断都落在物证
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。物证比对支撑体量。

## 第124章：西河渡空篓
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：一批空篓，可能比一辆车更能藏人
- **本章兑现点**：把西河渡堆场写成鲁伙计线的真实压力点
- **最晚冲突落点**：前25%
- **核心事件**：主角和邢管事探查西河渡堆场，发现空篓并非空篓，其中一只篓底有新割痕和血点
- **冲突类型**：堆场探查 / 人证逼近
- **人物发展**：主角从识破诱饵推进到逼近真实人证
- **信息释放**：鲁伙计可能被短时间藏在篓车或篓堆中转
- **钩子**：堆场管事忽然封场，说有官面临检
- **本章不可删场景**：空篓检查必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角追到西河渡空篓账
  - 必做事件：
    1. 探查堆场
    2. 找到新割痕和血点
    3. 堆场突然封场
  - 禁止越界：
    1. 不直接救出鲁伙计
    2. 不直接硬闯封场
  - 结束状态：主角确认已逼近真实转移点
  - 下章承接：围绕封场和官面临检进入反扑
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 堆场入场，负责立环境
    2. 空篓检查，负责拿到硬痕
    3. 封场反扑，负责拉高压力
  - 字数支撑理由：本章是现实探查章，动作充足
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。堆场现场能完整展开。

## 第125章：封场
- **功能**：压 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：主角刚摸到血点，对方就用官面封场，逼他从查线转为保线
- **本章兑现点**：对方反扑成立，主角必须保护已经拿到的空篓证据
- **最晚冲突落点**：前30%
- **核心事件**：堆场被官面临检封住，主角不硬闯，转而用空篓编号和血点位置先保住证据链
- **冲突类型**：官面反扑 / 保线取舍
- **人物发展**：主角从逼近人证转为保护证据链
- **信息释放**：对方已经知道他摸到西河渡，开始动用官面遮盖
- **钩子**：封场前，一个堆工偷偷塞给王兆陪半截篓绳，上面有鲁伙计的结法
- **本章不可删场景**：封场和保线必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：堆场管事宣布官面临检封场
  - 必做事件：
    1. 写出封场压力
    2. 主角保住空篓编号和血点线
    3. 堆工递出半截篓绳
  - 禁止越界：
    1. 不直接冲破官面封场
    2. 不直接救回鲁伙计
  - 结束状态：主角拿到鲁伙计仍可能活着的新证据
  - 下章承接：围绕篓绳结法追到真正藏人点
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 官面封场，负责反扑压力
    2. 证据保线，负责主角成熟动作
    3. 篓绳入手，负责下一章钩子
  - 字数支撑理由：本章是反扑与保线章，事件链清楚
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。封场和保线都有动作。

## 本批次备注
- **批次范围**：第121~125章
- **批次目标**：验证对方主动转移、假线诱导、第三地点反推和官面封场反扑是否能连续成立
- **批次风险点**：第123章容易写成纯推理，第125章容易写成纯总结；正文试写时必须保住物证和场景动作
- **对下一批的约束**：如果继续推进，下一批应围绕篓绳结法和真正藏人点，验证鲁伙计是否能从线索人物转回现实人物
