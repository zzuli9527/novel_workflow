# 新模具复跑细纲 v1：第136~140章

> 用途：承接第135章鲁伙计暂时脱险、不可公开和红泥来源线，验证救人后的反扑、藏身压力和调度权上抬是否连续成立。

## 批次目标

- 验证鲁伙计脱险后是否立刻带来新风险
- 验证对方反咬“私藏人证”是否能落成现实动作
- 验证红泥来源是否继续连接调度权线

## 批次风险点

- 写成救后修整
- 反扑不够强
- 红泥来源查得太顺

---

## 第136章：人不能露
- **功能**：压
- **目标字数**：2000~2500字
- **本章即时卖点**：鲁伙计救出半条命，却不能让任何人知道他活着
- **本章兑现点**：把救人后的藏身压力写实
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪和邢管事把鲁伙计转入临时藏身点，发现他身体虚弱且意识断续，无法直接作证
- **冲突类型**：藏人与护线 / 救后压力
- **人物发展**：主角意识到救人只是开始
- **信息释放**：鲁伙计公开露面会立刻被反咬成私藏人证
- **钩子**：外头开始传“王兆陪劫走仓役”的风声
- **本章不可删场景**：藏身点安置必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：鲁伙计暂时脱离押送，但不能公开
  - 必做事件：
    1. 安置鲁伙计
    2. 写出鲁伙计无法直接作证
    3. 外头出现私藏人证风声
  - 禁止越界：
    1. 不让鲁伙计恢复如常
    2. 不让常先生直接替主角压下风声
  - 结束状态：主角进入护人和抗反咬双重压力
  - 下章承接：围绕私藏人证反咬展开
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 藏身安置，负责救后压力
    2. 鲁伙计断续状态，负责限制口供
    3. 风声入城，负责引出反扑
  - 字数支撑理由：本章不是修整章，有藏身、状态限制和外部反咬三段
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。救后代价足够支撑正文。

## 第137章：谁劫了人证
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：对方不说鲁伙计失踪，反而说王兆陪劫了人证
- **本章兑现点**：让反咬成为现实压力，而非口头威胁
- **最晚冲突落点**：前30%
- **核心事件**：府城里开始有官面人查问“被劫仓役”，主角必须在不暴露鲁伙计的情况下稳住口径
- **冲突类型**：官面反咬 / 口径防守
- **人物发展**：主角第一次面对“救人被说成劫人”的反向压力
- **信息释放**：对方已经开始把鲁伙计从受害者改写成被劫走的仓役
- **钩子**：查问人拿出一份新写的“失役告文”
- **本章不可删场景**：官面查问必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：外头已有王兆陪劫人的风声
  - 必做事件：
    1. 官面查问出现
    2. 主角稳住口径
    3. 失役告文入手
  - 禁止越界：
    1. 不公开鲁伙计
    2. 不直接撕破官面
  - 结束状态：主角拿到对方反咬文本
  - 下章承接：围绕失役告文和红泥来源比对
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 风声扩大，负责立反咬压力
    2. 官面查问，负责现实冲突
    3. 失役告文，负责给证据入口
  - 字数支撑理由：反咬章有对抗场景，不靠解释
  - 场景级字数预算：
    1. 场景1：约600~750字
    2. 场景2：约850~1050字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。官面查问足够支撑。

## 第138章：红泥不是仓里的泥
- **功能**：争 / 破
- **目标字数**：2000~2500字
- **本章即时卖点**：失役告文上的封泥，和转篓牌上的红泥竟然同源
- **本章兑现点**：把红泥线与反咬文本接在一起
- **最晚冲突落点**：前30%
- **核心事件**：王兆陪比对红泥，确认它不属于仓里，而是来自府城一处旧印房常用泥料
- **冲突类型**：物证比对 / 来源追踪
- **人物发展**：主角学会把反咬文本反过来用作证据
- **信息释放**：对方反咬用的告文，反而露出调度口的泥源
- **钩子**：旧印房早已停用，却有人近期买过同类红泥
- **本章不可删场景**：红泥比对必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角拿到失役告文
  - 必做事件：
    1. 比对告文封泥与转篓牌红泥
    2. 确认旧印房泥源
    3. 发现近期购买痕迹
  - 禁止越界：
    1. 不直接找到购买者
    2. 不直接揭开调度者身份
  - 结束状态：红泥线指向旧印房
  - 下章承接：进入旧印房查购买痕迹
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 告文与红泥比对，负责物证连接
    2. 泥源确认，负责新地点建立
    3. 购买痕迹，负责下一章入口
  - 字数支撑理由：本章是物证比对章，动作清楚
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。红泥比对和泥源追踪足够展开。

## 第139章：旧印房
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：旧印房早已停用，却有人拿它的红泥给新告文封口
- **本章兑现点**：找到红泥来源的现实入口
- **最晚冲突落点**：前25%
- **核心事件**：主角和邢管事探查旧印房，发现近期有人借修门名义取走红泥，并留下内府杂役腰牌痕
- **冲突类型**：旧地探查 / 泥源追索
- **人物发展**：主角从反咬防守转向反查来源
- **信息释放**：调度权线可能牵到内府杂役系统
- **钩子**：腰牌痕对应的人，正好能自由出入府仓与旧印房
- **本章不可删场景**：旧印房探查必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：红泥线指向旧印房
  - 必做事件：
    1. 探查旧印房
    2. 找到近期取泥痕迹
    3. 发现内府杂役腰牌痕
  - 禁止越界：
    1. 不直接锁定最终调度者
    2. 不让常先生直接解谜
  - 结束状态：调度权线进入内府杂役层
  - 下章承接：围绕内府杂役出入权限继续追
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 旧印房入场，负责立废弃旧地
    2. 取泥痕迹，负责证实近期使用
    3. 腰牌痕，负责引出权限层
  - 字数支撑理由：本章是地点探查章，动作和发现明确
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。旧印房现场足够支撑正文。

## 第140章：能出入两边的人
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：真正可怕的不是一个杂役，而是有人能让杂役同时出入府仓和旧印房
- **本章兑现点**：把红泥来源推进到“出入权限”这个新层级
- **最晚冲突落点**：前35%
- **核心事件**：主角把失役告文、红泥、旧印房和腰牌痕并线，确认下一步要查的不是某个跑腿人，而是能安排出入权限的人
- **冲突类型**：阶段收束 / 权限目标建立
- **人物发展**：主角从查调度节点升级为查权限来源
- **信息释放**：调度权背后有一套能调动低层人手和旧物资源的权限链
- **钩子**：鲁伙计醒来后断续说出：“腰牌……不是他的。”
- **本章不可删场景**：权限并线和鲁伙计断续补证必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角拿到旧印房腰牌痕
  - 必做事件：
    1. 并线红泥与腰牌痕
    2. 建立出入权限目标
    3. 鲁伙计醒来补出半句
  - 禁止越界：
    1. 不直接找到权限背后的人
    2. 不让鲁伙计完全恢复
  - 结束状态：调度权线升级为权限链
  - 下章承接：围绕“腰牌不是他的”进入下一阶段
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 证据并线，负责阶段收束
    2. 权限目标建立，负责升级方向
    3. 鲁伙计补证，负责现实人证回压
  - 字数支撑理由：本章有并线、目标升级、人证补证三段
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。人证补证能避免纯总结。
