# 新模具复跑正文：第141~145章

> 用途：验证“腰牌不是他的”之后，借牌追查、人物试探、执行者反应和书办值房并线是否能连续写成正文。

## 第141章 腰牌不是他的
鲁伙计醒来的时间不长。

他只说了“腰牌不是他的”，便又开始喘。王兆陪原本还想追问，话到嘴边，却咽了回去。

不能逼。

人刚从死线上拖回来，若为了多问半句把他逼垮，前头救人的意义就少了一半。

王兆陪退出来，把旧印房的腰牌痕、红泥和失役告文重新排了一遍。腰牌不是他的，意思就很清楚：腰牌主人和使用者，可能不是同一个人。

他们去翻登记册时，很快看到一块牌。

那块牌当日记录完整得过分。

何时入，何时出，谁见过，谁交还，都写得干干净净。

干净得像有人怕它不干净。

## 第142章 登记太干净
登记册上的那块牌，确实找不出缺口。

可正因为找不出缺口，王兆陪才越看越觉得不对。

寻常腰牌出入，难免有笔迹轻重、交接拖延、门房补记。唯独这块牌，整整齐齐，像专门等人来查。

他没有先查缺漏，而是查这份完整是谁补出来的。

很快，问题出现了。

同一个时辰，这块牌的主人在府仓当值；可旧印房那边，也有同牌入门的痕迹。

牌只有一块。

人却像同时出现在两处。

这就不是牌主人能简单解释的事了。

## 第143章 腰牌主人
腰牌主人是个三十来岁的低级差役。

见到王兆陪时，他先是警惕，后来听王兆陪不问鲁伙计、不问旧印房，只问那日腰牌有没有离身，脸色才慢慢变了。

“王秀才问这个做什么？”

“有人用过你的牌。”王兆陪道。

对方沉默很久，手指一直抠着袖口。

王兆陪没有逼，也没有把鲁伙计那半句拿出来压他。他知道，这人未必是主谋，也未必完全干净。

最后，那差役压低声音说：“别问牌，问拿牌的人。”

说完这句，他转身就走，像怕自己再多说一个字，就会被什么东西盯上。

## 第144章 拿牌的人
拿牌的人不是官。

正因为不是官，才更方便。

王兆陪顺着腰牌主人留下的半句话，盯上了一名内府杂役。那人平日不起眼，常替人传话、领物、开偏门，若不是腰牌这条线，谁也不会把他和旧印房连在一处。

被盯的第二日，那杂役就察觉了。

他没有跑，也没有来找王兆陪，而是绕过府仓，去了书办值房方向。

这一下，线又往上抬了。

邢管事低声道：“要不要拿人？”

王兆陪摇头。

“拿他，只能拿一个跑腿的。跟着他，才能看他往谁那儿跑。”

## 第145章 书办值房
书办值房看着比府仓安静。

可越安静，越像能把话写成文、把人写成案的地方。

王兆陪没有进去，只从外头看那名杂役把话递进去。过了不久，值房里有人出来，手里拿着几页废稿，随手丢进后巷纸篓。

邢管事取回废稿时，纸还带着墨味。

其中一页，是失役告文的旧稿。词句和他们手里那份正式告文很像，只是更粗。

更要紧的是，另一页角落里有半行没写完的话：

“王兆陪私匿……”

后面的字被撕掉了。

王兆陪看着那半行字，心里反倒静了。

对方已经不只是遮鲁伙计。

他们开始把反扑直接写到他身上。
