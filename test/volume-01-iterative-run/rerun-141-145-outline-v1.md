# 新模具复跑细纲 v1：第141~145章

> 用途：承接第140章“腰牌不是他的”，验证借牌追查、腰牌主人与使用者分离、鲁伙计补证风险和权限链上抬是否连续成立。

## 批次目标

- 验证查腰牌时是否会暴露鲁伙计
- 验证借牌人是否形成现实对抗
- 验证权限链继续和红泥、告文、府仓三线绑定

## 批次风险点

- 写成查名单流程
- 借牌人没有反应
- 鲁伙计补证太完整，藏身压力减弱

---

## 第141章：腰牌不是他的
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：鲁伙计只说了半句，却把腰牌主人和使用者拆成了两个人
- **本章兑现点**：建立借牌追查方向
- **最晚冲突落点**：前30%
- **核心事件**：王兆陪在不继续逼问鲁伙计的情况下，根据半句补证和腰牌痕，判断有人借牌出入旧印房
- **冲突类型**：断续补证 / 借牌判断
- **人物发展**：主角学会保护人证，不为了查证硬逼鲁伙计
- **信息释放**：真正要查的不是腰牌名册，而是谁用过不属于自己的腰牌
- **钩子**：腰牌登记册上，某块牌当日记录完整得过分
- **本章不可删场景**：鲁伙计断续补证和主角停止追问必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：鲁伙计说出“腰牌不是他的”
  - 必做事件：
    1. 不继续逼问鲁伙计
    2. 判断腰牌主人和使用者分离
    3. 找到过分完整的登记记录
  - 禁止越界：
    1. 不让鲁伙计完整恢复
    2. 不直接找到借牌人
  - 结束状态：主角进入借牌追查
  - 下章承接：围绕登记册查腰牌主人
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 鲁伙计补证，负责承接
    2. 主角保护人证，负责人物成长
    3. 登记册异常，负责下一章入口
  - 字数支撑理由：本章不是查表，而是人证保护和证据判断并行
  - 场景级字数预算：
    1. 场景1：约600~750字
    2. 场景2：约750~950字
    3. 场景3：约500~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。人证保护提供场景张力。

## 第142章：登记太干净
- **功能**：争 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：最干净的登记，反而最不像真的
- **本章兑现点**：通过登记异常锁定腰牌主人
- **最晚冲突落点**：前30%
- **核心事件**：王兆陪查腰牌登记，发现目标腰牌当天出入记录过于完整，像是事后补齐
- **冲突类型**：登记核验 / 假干净识别
- **人物发展**：主角从查缺漏升级为查“太完整”
- **信息释放**：对方知道缺漏会被查，所以反向补出一份干净记录
- **钩子**：腰牌主人当天确实在府仓当值，却有人同时用他的牌进过旧印房
- **本章不可删场景**：登记册核验必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角发现某块腰牌登记过分完整
  - 必做事件：
    1. 核验登记册
    2. 识别过分干净
    3. 发现同牌双处出现
  - 禁止越界：
    1. 不直接认定腰牌主人有罪
    2. 不直接暴露鲁伙计补证
  - 结束状态：主角确认有人借牌
  - 下章承接：围绕腰牌主人试探
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 登记核验，负责查证
    2. 太干净判断，负责反推
    3. 同牌双处，负责钩子
  - 字数支撑理由：本章是核验章，但有反向判断和冲突
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。登记核验有过程。

## 第143章：腰牌主人
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：腰牌主人未必是坏人，但他一定知道自己的牌什么时候离过身
- **本章兑现点**：让腰牌主人从名单变成现实人物
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪不公开鲁伙计补证，借常规询问试探腰牌主人，发现他对当日一段空白极为回避
- **冲突类型**：人物试探 / 借牌回避
- **人物发展**：主角开始用不暴露底牌的方式问人
- **信息释放**：腰牌主人可能是被借、被胁或主动放牌
- **钩子**：腰牌主人低声提醒：“别问牌，问拿牌的人。”
- **本章不可删场景**：腰牌主人试探必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角确认同牌双处
  - 必做事件：
    1. 找到腰牌主人
    2. 不暴露鲁伙计补证地试探
    3. 腰牌主人给出“拿牌的人”提示
  - 禁止越界：
    1. 不直接逼供
    2. 不让腰牌主人全盘交代
  - 结束状态：借牌人开始浮出水面
  - 下章承接：围绕拿牌的人追查
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 找人，负责让名单变人物
    2. 试探，负责冲突
    3. 提醒，负责下一章入口
  - 字数支撑理由：本章是人物试探章，有对话和隐藏底牌
  - 场景级字数预算：
    1. 场景1：约600~750字
    2. 场景2：约850~1050字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。人物试探足够展开。

## 第144章：拿牌的人
- **功能**：破
- **目标字数**：2000~2500字
- **本章即时卖点**：拿牌的人不是高官，却能在关键时辰拿到牌，这才真正可怕
- **本章兑现点**：找到借牌执行者，并让对方产生反应
- **最晚冲突落点**：前30%
- **核心事件**：主角沿腰牌主人提示追到一名内府杂役，对方察觉被查后试图反向报信
- **冲突类型**：借牌执行者 / 反向报信
- **人物发展**：主角从查牌推进到逼执行者动起来
- **信息释放**：借牌执行者不是核心，但他能接触多个节点
- **钩子**：内府杂役报信对象不是府仓，而是一处书办值房
- **本章不可删场景**：借牌执行者反应必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角得到“拿牌的人”提示
  - 必做事件：
    1. 追到内府杂役
    2. 让对方察觉被查
    3. 跟到书办值房方向
  - 禁止越界：
    1. 不直接抓人审问
    2. 不直接揭开书办背后的人
  - 结束状态：权限链从杂役推进到书办值房
  - 下章承接：围绕书办值房建立下一阶段
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 追查杂役，负责找执行者
    2. 对方反应，负责让线活起来
    3. 报信去向，负责上抬节点
  - 字数支撑理由：本章有追踪和对方行动，不是名单章
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。对方反应提供动作量。

## 第145章：书办值房
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：权限链第一次摸到书办值房，说明调度权不只在脚夫和杂役手里
- **本章兑现点**：把借牌线推进到书办层
- **最晚冲突落点**：前35%
- **核心事件**：主角确认内府杂役向书办值房报信，同时发现书办值房近期处理过失役告文底稿
- **冲突类型**：节点上抬 / 文书源头
- **人物发展**：主角意识到文书、腰牌、红泥、府仓不是四条线，而是同一套调度写法
- **信息释放**：书办值房可能是反咬文本和权限调度的交汇点
- **钩子**：值房里一份废稿上，出现了王兆陪名字的半行草稿
- **本章不可删场景**：书办值房探查必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角跟到书办值房方向
  - 必做事件：
    1. 确认书办值房接收报信
    2. 发现失役告文底稿关联
    3. 找到带王兆陪名字的废稿
  - 禁止越界：
    1. 不直接找到最终主使
    2. 不直接公开鲁伙计
  - 结束状态：权限链和文书反咬线合并
  - 下章承接：围绕王兆陪名字草稿进入更直接的反扑
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 值房外观察，负责承接
    2. 底稿关联，负责并线
    3. 王兆陪名字废稿，负责强钩子
  - 字数支撑理由：本章是节点上抬章，有地点、文书、钩子三段
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。书办值房探查能支撑正文。
