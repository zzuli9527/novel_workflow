# 新模具复跑细纲 v1：第146~150章

> 用途：承接第145章“王兆陪私匿……”废稿，验证正式文书反咬、隐藏证人保护、实物证据反制和书办值房正面施压是否能连续成立。

## 批次目标

- 验证正式反咬是否能让主角处境变窄
- 验证主角是否能不公开鲁伙计而继续推进
- 验证废稿、旧稿、红泥、腰牌线能否合并成反证链
- 验证书办值房是否有持续对抗能力

## 批次风险点

- 写成解释前情
- 让书办值房一出手就露馅
- 主角靠口才脱身，证据链失去重量
- 文书证据过密，正文读感变成查档案

---

## 第146章：告文落名
- **功能**：压
- **目标字数**：2000~2500字
- **本章即时卖点**：废稿上的半句话，第二天变成盖印告文
- **本章兑现点**：正式反咬成立，王兆陪被迫接招
- **最晚冲突落点**：前20%
- **核心事件**：书办值房递出盖印告文，称王兆陪私匿失役人证、扰乱府仓清查，迫使王兆陪停止追查并交出线索
- **冲突类型**：正式文书施压 / 名声风险
- **人物发展**：主角从追查者转为被写进案中的当事人
- **信息释放**：对方不只是遮盖旧案，而是准备用官面文字反咬
- **钩子**：告文递送簿上，签收时辰比盖印时辰还早半刻
- **本章不可删场景**：盖印告文露面，主角被迫收文
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：王兆陪拿到写有自己名字的废稿
  - 必做事件：
    1. 正式告文送到
    2. 告文把王兆陪写成私匿人证者
    3. 王兆陪发现递送时辰漏洞
  - 禁止越界：
    1. 不公开鲁伙计
    2. 不当场翻案
    3. 不让对方降智承认
  - 结束状态：主角被文书压住，但发现时辰漏洞
  - 下章承接：查递送路径
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 告文送达，负责压迫
    2. 主角收文应对，负责人物反应
    3. 递送簿时辰异常，负责反制入口
  - 字数支撑理由：主角由追查转为被反咬，现实压力和证据观察并行
  - 场景级字数预算：
    1. 场景1：约650~850字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。告文压人具备强冲突。

## 第147章：早了半刻
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：公文还没盖好，递送簿已经先写了签收
- **本章兑现点**：递送路径出现漏洞
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪借核对签收为由查递送簿，发现告文从书办值房流出前，签收记录已提前写好
- **冲突类型**：递送核验 / 预写记录
- **人物发展**：主角不急着辩白，改查文书运行路径
- **信息释放**：反咬告文不是临时发现，而是预设好的套
- **钩子**：提前签收的那一笔，墨色和废稿上的“私匿”二字一样新
- **本章不可删场景**：递送簿核验必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角发现签收时辰早于盖印
  - 必做事件：
    1. 查递送簿
    2. 确认时辰预写
    3. 发现墨色关联
  - 禁止越界：
    1. 不把递送小吏写成主动招供
    2. 不直接拿废稿当公开证据
  - 结束状态：主角确认反咬告文预制
  - 下章承接：查废稿、旧稿和红泥痕
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 核对签收，负责动作
    2. 递送小吏回避，负责对抗
    3. 墨色和废稿关联，负责钩子
  - 字数支撑理由：递送路径能写出人和物，不只是查表
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约500~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。需要靠小吏回避增强现实感。

## 第148章：同一滴红泥
- **功能**：争 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：旧稿、废稿和正式告文，不该有同一处红泥痕
- **本章兑现点**：废稿、旧稿、红泥线合并
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪比对旧稿、废稿和正式告文，发现三者边角都有同样的红泥拖痕，说明它们曾在同一处案板上被处理
- **冲突类型**：实物比对 / 证据合并
- **人物发展**：主角把散线合成证据链，但仍忍住不公开
- **信息释放**：书办值房和旧印房之间存在同一文书处理点
- **钩子**：能留下这种红泥痕的案板，只在值房后间
- **本章不可删场景**：三份文书比对必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角拿着正式告文、废稿、旧稿
  - 必做事件：
    1. 比对三份纸
    2. 发现同一处红泥拖痕
    3. 锁定值房后间案板
  - 禁止越界：
    1. 不直接冲进值房拿人
    2. 不让证据完全闭合
  - 结束状态：主角掌握文书处理地点
  - 下章承接：书办值房反压邢管事
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 三稿同桌，负责证据观察
    2. 红泥痕复核，负责推理
    3. 值房后间，负责下一步入口
  - 字数支撑理由：实物证据合并能形成细节快感
  - 场景级字数预算：
    1. 场景1：约600~800字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。红泥细节必须写清楚。

## 第149章：先拿邢管事
- **功能**：压
- **目标字数**：2000~2500字
- **本章即时卖点**：对方不抓王兆陪，先动给他递纸的人
- **本章兑现点**：书办值房形成更强现实压力
- **最晚冲突落点**：前20%
- **核心事件**：书办值房以“擅取废稿”为名扣住邢管事，逼王兆陪交出废稿来源和鲁伙计线索
- **冲突类型**：同伴受压 / 证据来源风险
- **人物发展**：主角意识到证据来源本身也会变成弱点
- **信息释放**：对方知道废稿流出，开始反查主角身边人
- **钩子**：邢管事被带走前，只来得及留下一句“案板换了”
- **本章不可删场景**：邢管事被扣和主角不能立刻救人必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角锁定值房后间案板
  - 必做事件：
    1. 邢管事被扣
    2. 对方逼问废稿来源
    3. 邢管事留下“案板换了”
  - 禁止越界：
    1. 不让主角直接硬闯救人
    2. 不让邢管事交代鲁伙计
  - 结束状态：主角失去一个外部帮手
  - 下章承接：追查被换走的案板
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 值房扣人，负责压力
    2. 主角被迫权衡，负责人物
    3. 案板换走，负责钩子
  - 字数支撑理由：同伴被压让文书战有现实代价
  - 场景级字数预算：
    1. 场景1：约750~900字
    2. 场景2：约750~950字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。外部压力足够支撑章节。

## 第150章：换走的案板
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：案板被换走了，却在板脚留下最后一点红泥
- **本章兑现点**：主角拿到可反制但不能立刻公开的关键证据
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪追查被换走的案板，发现旧案板被送到废库，板脚残留红泥和纸屑，能证明三份文书同案处理，但公开它会暴露邢管事取废稿和鲁伙计藏身线
- **冲突类型**：证据取得 / 公开代价
- **人物发展**：主角学会拿到证据后仍要等合适时机
- **信息释放**：真正能反制的不是一张纸，而是文书流转场所和处理痕迹
- **钩子**：废库外有人先一步点了灯，像在等他上门
- **本章不可删场景**：找到旧案板和不能公开的权衡必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：邢管事被扣，案板被换
  - 必做事件：
    1. 追查旧案板去向
    2. 找到红泥和纸屑残留
    3. 判断证据不能立刻公开
  - 禁止越界：
    1. 不立刻翻盘
    2. 不让鲁伙计公开出场
    3. 不让邢管事立刻脱困
  - 结束状态：主角拿到关键反证，但被人堵在废库
  - 下章承接：废库正面对抗或反设局
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 查案板去向，负责动作
    2. 废库取证，负责兑现
    3. 灯火堵门，负责下章钩子
  - 字数支撑理由：取得证据和无法公开之间有强拉扯
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。证据取得和被堵门构成双峰。
