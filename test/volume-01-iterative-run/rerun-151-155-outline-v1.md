# 新模具复跑细纲 v1：第151~155章

> 用途：承接第150章“废库外有人点灯堵门”，验证堵门者是否能带出新压力，旧案板残证是否能在不公开鲁伙计和邢管事线的情况下被保住，并测试 `04-draft.md` 的“证据公开代价检查”是否有效。

## 批次目标

- 验证废库堵门不是普通抓人，而是确认主角拿到哪份残证
- 验证主角能否保住旧案板残证，同时避免暴露证据来源
- 验证正式文书反咬继续压住主角，不在本批解决
- 验证邢管事被扣的压力仍然有效

## 批次风险点

- 堵门写成抓现行
- 主角立刻亮证据翻盘
- 邢管事过快脱困
- 证据拆分只剩机械操作，缺少人物压力

---

## 第151章：灯在门外
- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：堵门的人不开口抓他，只问他手里到底拿了哪一份
- **本章兑现点**：堵门者带出交易与确认残证范围的压力
- **最晚冲突落点**：前20%
- **核心事件**：废库外点灯的人是书办值房的曹书吏，他不搜王兆陪，只提出用邢管事换竹筒，逼王兆陪判断对方到底知道多少
- **冲突类型**：堵门试探 / 证据交换
- **人物发展**：主角从“拿到证据”转入“保护证据信息差”
- **信息释放**：对方不知道竹筒里具体是什么，只想确认主角拿到了哪份残证
- **钩子**：曹书吏说邢管事今晚会被改成“自认擅取文书”
- **本章不可删场景**：废库堵门和交易试探必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：王兆陪拿着竹筒，被人堵在废库
  - 必做事件：
    1. 曹书吏点灯堵门
    2. 对方提出用邢管事换竹筒
    3. 主角判断对方不确定竹筒内容
  - 禁止越界：
    1. 不公开竹筒残证
    2. 不救出邢管事
    3. 不暴露鲁伙计
  - 结束状态：主角守住竹筒，但邢管事供词风险升级
  - 下章承接：主角需要用半真半假方式脱身
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 灯火堵门，负责压迫
    2. 交易试探，负责对抗
    3. 邢管事供词威胁，负责钩子
  - 字数支撑理由：对话、心理判断和现实压力并行
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。堵门对抗有明确张力。

## 第152章：半截纸屑
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪交出去一份看似关键、实则不完整的残证
- **本章兑现点**：半真半假的证据应对成立
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪把无“匿”字的纸屑边角交出去，保住真正带残墨的半截，并借对方反应确认值房最怕的是残墨字迹而不是红泥
- **冲突类型**：证据拆分 / 反应试探
- **人物发展**：主角开始主动利用对方的不确定
- **信息释放**：对方怕的不是案板红泥本身，而是能和“私匿”废稿对应的残墨
- **钩子**：曹书吏收下假残证后，仍命人去查王兆陪身边的“写字人”
- **本章不可删场景**：半真半假交证和对方反应必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角需要从废库脱身，同时保住关键残墨
  - 必做事件：
    1. 拆分残证
    2. 交出不完整部分
    3. 通过对方反应确认残墨才是核心
  - 禁止越界：
    1. 不把真残墨交出
    2. 不靠嘴压人脱身
    3. 不让对方完全被骗
  - 结束状态：主角暂时脱身，但对方转查写字人
  - 下章承接：邢管事被逼认“擅取文书”
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 证据拆分，负责动作
    2. 交出半截，负责对抗
    3. 对方转查写字人，负责钩子
  - 字数支撑理由：证据处理有动作，反应试探有张力
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。关键是写出对方反应。

## 第153章：擅取文书
- **功能**：压
- **目标字数**：2000~2500字
- **本章即时卖点**：邢管事不认私藏人证，却被逼认擅取文书
- **本章兑现点**：书办值房继续压住证据来源
- **最晚冲突落点**：前20%
- **核心事件**：值房把邢管事的供词改成“擅取废稿”，不碰鲁伙计，却足以切断王兆陪的废稿来源
- **冲突类型**：供词改写 / 证据来源反咬
- **人物发展**：主角意识到对方不需要证明全部罪名，只需污染证据来源
- **信息释放**：正式反咬开始从主角本人转向证据链可信度
- **钩子**：供词末尾多出一句“废稿交与王兆陪后，另有人誊写”
- **本章不可删场景**：邢管事受压和供词被污染必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：对方转查写字人，邢管事仍被扣
  - 必做事件：
    1. 主角得知邢管事供词被改
    2. 供词锁定“擅取文书”
    3. 末尾引出“另有人誊写”
  - 禁止越界：
    1. 不救出邢管事
    2. 不让邢管事全盘翻供
    3. 不公开鲁伙计
  - 结束状态：证据来源被污染，写字人线出现
  - 下章承接：查“堵门不是抓人，而是确认残证”
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 供词传出，负责压力
    2. 主角判断证据来源被污染，负责人物
    3. 写字人线出现，负责钩子
  - 字数支撑理由：供词反咬把文书线落到人物风险上
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约850~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。外部压力清晰。

## 第154章：他们要看的不是板
- **功能**：破 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：对方昨夜堵门，不是为了抓他，而是为了看他到底护着哪一种证据
- **本章兑现点**：主角识破废库堵门的真实目的
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪复盘曹书吏的问法和后续追查，确认对方真正要找的是带“匿”字残墨的半截纸屑，以及可能誊写过废稿的人
- **冲突类型**：目的识别 / 反向判断
- **人物发展**：主角从应对堵门转为反查对方恐惧点
- **信息释放**：书办值房害怕残墨连接正式告文和废稿
- **钩子**：写字人不是外人，而是值房内负责誊清的瘦书办
- **本章不可删场景**：识破堵门目的必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：证据来源被污染，写字人线出现
  - 必做事件：
    1. 复盘曹书吏堵门问法
    2. 确认对方怕残墨字迹
    3. 锁定值房内瘦书办
  - 禁止越界：
    1. 不直接拿瘦书办
    2. 不让瘦书办招供
    3. 不公开残墨
  - 结束状态：主角知道该反查谁，但还缺保护残证的方法
  - 下章承接：残证拆成两份保管
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 复盘堵门，负责推理
    2. 追查写字人，负责动作
    3. 瘦书办浮出，负责钩子
  - 字数支撑理由：不是空想复盘，有外部追查和人物浮出
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约500~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。需防止纯推理化。

## 第155章：一证两藏
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪把同一份残证拆成“能引人追的假线”和“不能见光的真线”
- **本章兑现点**：主角保住残证，并换来反查瘦书办的机会
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪把红泥拓痕和带“匿”字纸屑分开保管，用红泥拓痕引曹书吏继续误判，暗中保住残墨纸屑，并借假线逼瘦书办露面

- **人物发展**：主角学会让证据暂时不见光，却能推动局势
- **信息释放**：证据使用不只有公开一种方式，还能诱导对方动作
- **钩子**：瘦书办夜里去了旧印房，手里拿着一张新誊的空白告文
- **本章不可删场景**：一证两藏和瘦书办露面必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角锁定瘦书办，但残证不能公开
  - 必做事件：
    1. 残证拆分保管
    2. 用红泥拓痕制造假线
    3. 瘦书办被假线逼出动作
  - 禁止越界：
    1. 不解决正式文书反咬
    2. 不救出邢管事
    3. 不公开鲁伙计
  - 结束状态：主角获得反查瘦书办的机会
  - 下章承接：旧印房新告文

    3. 瘦书办夜去旧印房，负责钩子
