# 新模具复跑细纲 v1：第156~160章

> 用途：承接第155章“瘦书办夜入旧印房，新誊空白告文”，验证新告文压力、红泥拓痕假线、对手误判状态和真残墨保护是否能连续成立。

## 批次目标

- 验证瘦书办不能立刻招供，而要继续制造文书压力
- 验证新誊空白告文不是解释旧案，而是压出新的现实风险
- 验证曹书吏对红泥拓痕假线的误判能继续驱动对手行动
- 验证 `05-update-state.md` 的“对手认知 / 误判状态”能作为下一章输入

## 批次风险点

- 瘦书办变成低级工具人
- 新告文只服务解释旧案
- 主角通过真残墨直接翻盘
- 对手误判状态只写成总结，没有推动下一章

## 第156章：门板旧痕
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：瘦书办不是来毁证，而是在旧印房门板上找能复刻的新格式
- **本章兑现点**：新告文制造压力的方向出现
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪跟到旧印房，发现瘦书办用新誊空白告文比对门板旧痕，像在复刻另一份可盖印的反咬文本
- **冲突类型**：跟踪观察 / 新文书成形
- **人物发展**：主角从追查旧文书转为防新文书成案
- **信息释放**：瘦书办掌握的不是全部真相，而是能把旧痕转成新告文格式的手路
- **钩子**：新告文空着的人名位置，大小刚好能写下“常先生”
- **本章不可删场景**：旧印房门板比对和新告文人名空位必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：瘦书办夜入旧印房，手里有新誊空白告文
  - 必做事件：
    1. 主角跟踪瘦书办
    2. 瘦书办比对旧印房门板旧痕
    3. 新告文人名空位指向常先生风险
  - 禁止越界：
    1. 不让瘦书办招供
    2. 不公开真残墨
    3. 不解决正式反咬
  - 结束状态：主角确认新告文可能压向常先生
  - 下章承接：曹书吏继续追红泥拓痕假线
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 夜跟瘦书办，负责动作
    2. 门板旧痕比对，负责信息
    3. 常先生人名空位，负责钩子
  - 字数支撑理由：跟踪、观察和新风险能形成双峰
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。新文书风险明确。

## 第157章：假线被咬住
- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：曹书吏咬住红泥拓痕，却把真正的残墨放过了
- **本章兑现点**：红泥拓痕假线继续影响对手行动
- **最晚冲突落点**：前25%
- **核心事件**：曹书吏派人追查红泥拓痕来源，误以为王兆陪准备公开案板证据，因此开始调走与案板有关的小吏和器物
- **冲突类型**：假线诱导 / 对手误判行动
- **人物发展**：主角开始利用对手误判争取时间
- **信息释放**：曹书吏仍认为红泥拓痕是主证，而不知道真残墨已外转
- **钩子**：被调走的小吏中，有一人正是递送簿提前签收那天的值手
- **本章不可删场景**：假线被咬住和小吏调走必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角知道新告文可能压向常先生
  - 必做事件：
    1. 曹书吏追红泥拓痕
    2. 对方调走案板相关小吏
    3. 提前签收值手被卷入
  - 禁止越界：
    1. 不公开真残墨
    2. 不让曹书吏发现假线
    3. 不直接拿值手
  - 结束状态：主角确认假线有效，同时发现旧签收值手被调走
  - 下章承接：新告文压力引向常先生
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 曹书吏查拓痕，负责压力
    2. 小吏被调走，负责现实动作
    3. 值手身份露出，负责钩子
  - 字数支撑理由：对手误判带来实际调动，不是纯总结
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约500~650字
  - 摘要风险判断：中
  - 最低可撑体量判断：能。需写出对手动作。

## 第158章：常先生入文
- **功能**：压
- **目标字数**：2000~2500字
- **本章即时卖点**：新告文没有先写王兆陪，而是先把常先生写成“藏证授意者”
- **本章兑现点**：新告文制造新的现实风险
- **最晚冲突落点**：前20%
- **核心事件**：王兆陪得知新告文草稿把常先生写成藏证授意者，逼他权衡是否取回真残墨，若取回会暴露保管线，若不取回常先生会被拖入文书反咬
- **冲突类型**：外部保管线受压 / 新告文威胁
- **人物发展**：主角面对保护证据与保护人的冲突
- **信息释放**：对方已经从红泥假线推测出王兆陪有外部保管人，但对象还未完全确定
- **钩子**：常先生收到一封没有落款的短笺，只写“还筒，免祸”
- **本章不可删场景**：常先生入文和取回真残墨的权衡必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：假线有效，但值手被调走
  - 必做事件：
    1. 新告文草稿指向常先生
    2. 主角权衡是否取回真残墨
    3. 常先生收到“还筒，免祸”短笺
  - 禁止越界：
    1. 不公开真残墨
    2. 不让常先生公开出面作证
    3. 不解决正式反咬
  - 结束状态：常先生保管线被压住
  - 下章承接：主角用对手误判反设小局
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 新告文消息传来，负责施压
    2. 是否取回真残墨，负责人物取舍
    3. 短笺威胁，负责钩子
  - 字数支撑理由：证据与人物保护冲突清晰
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约850~1000字
    3. 场景3：约450~650字
  - 摘要风险判断：低
  - 最低可撑体量判断：能。人物压力充足。

## 第159章：空筒回去

- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪让对方看到“筒回来了”，却没让真正的残墨回来

- **最晚冲突落点**：前25%
- **核心事件**：王兆陪不取回真残墨，只让常先生送回空竹筒和一片新刮红泥，诱使曹书吏误判主角已把证据收回府内，从而暴露监视常先生的人

- **钩子**：茶摊掌柜向瘦书办递话：“筒回来了，人没动。”
- **本章不可删场景**：空筒回府和监视者暴露必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：常先生保管线被压住
  - 必做事件：
    1. 不取回真残墨
    2. 空筒回府制造误判
    3. 监视常先生的人暴露
  - 禁止越界：
    1. 不让真残墨回府
    2. 不让常先生公开对抗
    3. 不抓茶摊掌柜
  - 结束状态：主角确认府外监视线与瘦书办相连
  - 下章承接：瘦书办暴露新告文真正人名
- **章节可写性证明**：

    2. 监视者露头，负责动作

- **本章不可删场景**：人名反转和邢管事风险升级必须落地

- **节奏**：双峰

    2. 发现目标不是常先生而是邢管事

    3. 不公开真残墨
