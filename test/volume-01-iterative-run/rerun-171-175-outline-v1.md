# 新模具复跑细纲 v1：第171~175章

> 用途：承接第170章“改过的签押样本背面有疑案急请封泥痕”，验证签押真假验证是否能进入现场动作，并继续压住王某书生补名风险。

## 批次目标

- 验证签押真假验证不退化成纸面推理
- 验证原签押册和封泥来源都能落到现场动作
- 验证签押样本只能证明“样本不干净”，不能直接翻盘
- 验证王某书生补名压力仍然存在

## 批次风险点

- 签押真假写成技术鉴定说明
- 原签押册直接给出答案
- 瘦书办或曹书吏降智暴露
- 忘记上级回批仍在逼近

## 第171章：签押册不在
- **功能**：争
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪去找原签押册，册子却刚被调去抄房
- **本章兑现点**：签押验证进入现场动作
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪按签押样本去找原签押册，发现存册架上空了一格，管册小吏称册子被临时调去抄房核对王某身份
- **冲突类型**：找册现场 / 临时调册
- **人物发展**：主角从手握样本转为追原册流向
- **信息释放**：对手已经抢先一步动原签押册
- **钩子**：管册小吏悄悄提醒：“他们只调走了王姓那一册。”
- **本章不可删场景**：存册架空格和管册小吏提醒必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角手握改过的签押样本，但还不能证明是谁改
  - 必做事件：
    1. 去存册处找原签押册
    2. 发现王姓册被调去抄房
    3. 管册小吏提醒只调了王姓册
  - 禁止越界：
    1. 不直接拿到完整答案
    2. 不公开真残墨
    3. 不解除王某书生压力
  - 结束状态：主角必须追到抄房找册
  - 下章承接：原签押册缺页
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 存册架找册，负责现场动作
    2. 小吏回避，负责对抗
    3. 只调王姓册，负责钩子
  - 字数支撑理由：找册、回避和追册流向能展开为现场对抗
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：签押验证落到存册现场和调册后果
  - 最低可撑体量判断：能。现场找册和小吏回避足够展开。
