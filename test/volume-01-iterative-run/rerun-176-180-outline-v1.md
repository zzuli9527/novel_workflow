# 新模具复跑细纲 v1：第176~180章

> 用途：验证纸纤维、洗字水、纸坊账这类“证据链章节”能否继续保持现场动作，而不是退回技术说明。

## 批次目标

- 验证纸纤维线索能否落到纸料账和纸坊现场
- 验证洗字水线索能否落到换盆、倒水和残渣现场
- 验证半证只能拖住补名，不能直接翻盘
- 验证王某疑名持续制造压迫

## 批次风险点

- 纸纤维变成技术鉴定
- 洗字水直接恢复原字
- 曹书吏压力消失
- 瘦书办当场露底

---

## 第176章：纸料账
- **功能**：争 / 查
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪拿着旧纸纤维去找纸料账，发现王姓签押册用纸不是普通公文纸
- **本章兑现点**：纸纤维验证进入纸料账现场
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪去库账房查存册纸料，账房管事先推说旧账封存，后因账角编号露出同批纸料记录，确认王姓旧签押册来自固定纸坊
- **冲突类型**：查账受阻 / 半证推进
- **人物发展**：主角从拿到纤维转为追纸料来源
- **信息释放**：王姓签押册用的是一批带麻丝的旧纸，不是随手可替换的新纸
- **钩子**：账房管事说这批纸料后来又被抄房领过一叠
- **本章不可删场景**：库账房查账、账角编号、纸料批次必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角掌握旧纸纤维，但不能证明原字
  - 必做事件：
    1. 去库账房查纸料账
    2. 账房管事阻拦
    3. 查出王姓册纸料来自固定纸坊
  - 禁止越界：
    1. 不复原原签押
    2. 不让账房管事完整招供
    3. 不公开真残墨
  - 结束状态：主角锁定纸坊和抄房领纸记录
  - 下章承接：回查洗笔房水盆
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 库账房找账，负责现场动作
    2. 账角编号对照，负责信息推进
    3. 抄房领纸记录，负责钩子
  - 字数支撑理由：查账阻拦、编号对照和领纸记录能形成现场对抗
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~950字
    3. 场景3：约550~700字
  - 摘要风险判断：中
  - 流程章落地检查：纸纤维落到库账房现场
  - 最低可撑体量判断：能。查账和领纸记录足够支撑。

## 第177章：水盆空了
- **功能**：压 / 查
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪回洗笔房查洗字水，水盆已经被人倒空
- **本章兑现点**：洗字水不能直接给答案，只留下残渣半证
- **最晚冲突落点**：前20%
- **核心事件**：洗笔房水盆被倒，杂役正换盆，王兆陪只从盆缝和地沟边截下纸浆残渣，发现有人抢在他前面清洗现场
- **冲突类型**：灭迹现场 / 残证截取
- **人物发展**：主角接受证据被削弱，继续拿半证推进
- **信息释放**：洗字水确实被处理过，但原字无法恢复
- **钩子**：倒水杂役说是“瘦先生吩咐换清水”
- **本章不可删场景**：洗笔房换盆、地沟残渣必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角知道纸料批次，回查洗字水
  - 必做事件：
    1. 洗笔房水盆已空
    2. 杂役换盆并试图冲掉残渣
    3. 主角截下盆缝纸浆
  - 禁止越界：
    1. 不恢复原字
    2. 不让杂役知道全局
    3. 不让瘦书办当场认错
  - 结束状态：主角得到洗字残渣，但仍是半证
  - 下章承接：去纸坊验证纸浆来源
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 回到洗笔房，负责冲突落点
    2. 抢水换盆，负责动作
    3. 截取残渣，负责线索
  - 字数支撑理由：灭迹现场和残证截取能支撑动作和心理压力
  - 场景级字数预算：
    1. 场景1：约600~750字
    2. 场景2：约850~1000字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：洗字水落到换盆和倒水现场
  - 最低可撑体量判断：能。

## 第178章：纸坊旧账
- **功能**：查 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：纸坊旧账证明同批纸只供过存册处和抄房
- **本章兑现点**：纸纤维半证变成可拖补名的流程证据
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪到纸坊查旧账，纸匠认出带麻丝的旧纸，并说同批纸后来被府里补领一叠，领纸牌不是存册处而是抄房
- **冲突类型**：外部取证 / 账册对照
- **人物发展**：主角从府内追查扩展到府外纸坊
- **信息释放**：缺页纸料来源更接近原册，但仍不能证明原字内容
- **钩子**：纸匠记得领纸人手上有洗墨水的灰痕
- **本章不可删场景**：纸坊查账和纸匠认纸必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角拿到纸浆残渣和纸料批次线索
  - 必做事件：
    1. 到纸坊查旧账
    2. 纸匠认出带麻丝旧纸
    3. 查到抄房补领同批纸
  - 禁止越界：
    1. 不证明原签押文字
    2. 不直接锁死瘦书办
    3. 不让纸坊账成为终局铁证
  - 结束状态：主角取得同批旧纸供给记录
  - 下章承接：曹书吏逼补名，主角用半证拖住
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 纸坊到访，负责现场
    2. 纸匠认纸，负责半证
    3. 旧账对照，负责推进
  - 字数支撑理由：纸坊人情、旧账对照和领纸人细节能撑开
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~900字
    3. 场景3：约600~750字
  - 摘要风险判断：中
  - 流程章落地检查：纸纤维验证落到纸坊人物和账册
  - 最低可撑体量判断：能。

## 第179章：半证挡名
- **功能**：争 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：曹书吏催补王某全名，王兆陪拿纸料半证只换来暂缓，不是洗清
- **本章兑现点**：半证只能拖压，不能翻盘
- **最晚冲突落点**：前20%
- **核心事件**：府里开补名会，曹书吏要求把王兆陪名字补进疑案急请附注；王兆陪提交纸料来源疑点，迫使补名暂缓，但王某疑名继续挂着
- **冲突类型**：会场对抗 / 流程拖延
- **人物发展**：主角更明确“不求现在赢，只求不被写死”
- **信息释放**：纸料半证足以证明签押样本被动过，但不足以证明谁动、为何动
- **钩子**：曹书吏要求明日拿出“洗字人”，否则继续补名
- **本章不可删场景**：补名会场和半证递交必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角拿到纸坊旧账和纸料半证
  - 必做事件：
    1. 曹书吏逼补王兆陪全名
    2. 主角提交纸料疑点
    3. 补名暂缓但王某疑名保留
  - 禁止越界：
    1. 不撤掉王某疑名
    2. 不当场定瘦书办罪
    3. 不让曹书吏败退
  - 结束状态：主角多争一天，但被要求找出洗字人
  - 下章承接：纸纤维结论无法复原原字
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 补名会开场，负责压力
    2. 半证递交，负责对抗
    3. 一日限期，负责钩子
  - 字数支撑理由：会场争辩和流程后果能撑开
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：半证进入补名会场并产生人物后果
  - 最低可撑体量判断：能。

## 第180章：只能证明纸
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：纸纤维结论成立，却只证明缺页来自原册，不能证明原字
- **本章兑现点**：证据公开代价检查生效
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪把纸纤维、纸坊旧账和残渣对齐，确认缺页来自原册纸料；但原字被洗淡重描，无法恢复，曹书吏借此要求继续查洗字人
- **冲突类型**：证据边界 / 下一目标转移
- **人物发展**：主角接受“只赢半步”的状态
- **信息释放**：下一批必须追谁拿了旧纸、谁倒了洗字水
- **钩子**：洗笔房杂役忽然改口，说倒水时看见的是“戴黑袖套的人”，不是瘦书办本人
- **本章不可删场景**：证据对齐和结论边界必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：补名暂缓一日，必须给出洗字线索
  - 必做事件：
    1. 对齐纸纤维、纸坊账、洗字残渣
    2. 证明缺页来自原册纸料
    3. 承认无法复原原字
  - 禁止越界：
    1. 不洗清主角
    2. 不暴露真残墨
    3. 不确认瘦书办就是唯一执行人
  - 结束状态：主角获得纸料半证，下一批转向洗字人和黑袖套
  - 下章承接：追查倒水人和黑袖套
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 证据对齐，负责收束
    2. 原字不可复原，负责代价
    3. 黑袖套改口，负责新钩子
  - 字数支撑理由：半证边界、流程后果和新疑点能撑开
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约750~900字
    3. 场景3：约550~700字
  - 摘要风险判断：中
  - 流程章落地检查：证据边界落到补名后果和新追查对象
  - 最低可撑体量判断：能。
