# 第181~185章批次评审（v1）

## 一、总体判断

第181~185章基本通过“黑袖套不能空转”的验证。黑袖套没有停留在抽象猜谜，而是依次落到值房实物、夜里领纸、代领牌、残线、湿灰水痕和门禁空押。

这一批的关键验证点成立：

1. 黑袖套进入具体场所：值房、库房、抄房、洗笔房
2. 一日限期没有消失，而是被压缩到午后
3. 灰墨水痕与领纸人、湿灰水、洗字水形成关联
4. 黑袖套没有直接终结瘦书办线，只证明有人夜进洗笔房
5. 王某疑名仍然保留，曹书吏继续推动补名

## 二、单章结果

- 第181章：黑袖套从模糊特征落到夜值共用护袖
- 第182章：灰墨水痕与代领旧纸人挂钩，但代领人不是终点
- 第183章：一日限期压缩为午后，值房夹板被取走
- 第184章：残线和湿灰水成立，但袖套多人共用
- 第185章：黑袖套只能证明有人夜进洗笔房，下一批转向门禁空押

## 三、本批主要问题

### 1. 黑袖套没有空转

本批每章都让黑袖套产生新状态：

- 181：确认它是共用护袖
- 182：确认缺系绳细节和代领纸关联
- 183：值房夹板被取走，产生证据损失
- 184：找到残线和湿灰水痕
- 185：转化为“有人夜进洗笔房”的阶段结论

黑袖套从“猜是谁”变成“查共用物如何流转”，符合本轮要求。

### 2. 限期压力有效

曹书吏没有退场，也没有被半证压倒。限期从“一日”变成“午后”，让主角的查证带有代价。

这一点说明连续长跑中“压力不能消失”的规则仍然有效。

### 3. 下一批要避免“门禁册继续被动挪证据”

门禁空押是新线索，但如果继续写，很容易又变成“去查一本册子，发现又被拿走”。下一批必须要求：

- 门禁空押要牵出具体值门人
- 夜值牌和门禁记录不能同时被抽走
- 至少有一个现场人物要承担后果
- 午后补名压力继续推进，不能靠查册无限延期

## 四、对工作流的修订结论

本批暴露出一个轻微模具风险：连续证据链章节容易重复使用“记录被取走 / 账册被调走 / 夹板被拿走”的阻断方式。

这不是当前样本的硬失败，但如果继续迭代，工作流应补充一条约束：同一批次内，证据阻断方式不能连续重复；每次阻断必须改变追查方向或造成具体人物后果。

## 五、建议修订

建议在 `workflow/03-chapters.md` 的流程章或连续长跑约束中补充：

- 同一批次内不得连续用“册子被调走、记录被取走、账本被封存”作为主要阻断
- 若使用文书 / 记录阻断，必须同步给出：
  1. 具体执行人或受影响人
  2. 主角损失的时间、机会或证据
  3. 新追查方向

## 六、下一批约束

如果继续往下写，重点观察：

1. 门禁空押必须牵出值门人
2. 夜值牌和门禁记录不能同时被抽走
3. 午后补名压力继续存在
4. 王某疑名不能消失
5. 至少一个现场人物必须承担后果
