# 新模具复跑细纲 v1：第181~185章

> 用途：验证“黑袖套”线索能否从模糊特征落到具体人物、具体场所和具体流程后果，避免悬疑化空转。

## 批次目标

- 验证黑袖套是否能落到值房、库房、抄房等具体场所
- 验证一日限期是否持续压迫王兆陪
- 验证灰墨水痕是否能与领纸人、倒水人或洗字水形成现场关联
- 验证黑袖套只能推进追查，不能直接终结瘦书办线

## 批次风险点

- 黑袖套成为单一谜面，连续几章只猜不动
- 一日限期被遗忘
- 代领人直接招供导致线索过快收束
- 王某疑名压力消失

---

## 第181章：袖套在哪
- **功能**：查 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪查黑袖套来源，发现它不是某一人的私物，而是府里夜值用具
- **本章兑现点**：黑袖套落到具体场所和值用物
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪先查洗笔房杂役所见黑袖套，再去值房和库房对照，发现黑袖套在库房搬箱、抄房夜值、封袋处护袖中都出现过
- **冲突类型**：线索扩散 / 时间压迫
- **人物发展**：主角从追单人转为追共用物和使用记录
- **信息释放**：黑袖套不是直接指向瘦书办，而是指向夜间能进出多个房口的人
- **钩子**：值房旧架上少了一副黑袖套
- **本章不可删场景**：值房查袖套、库房对照、缺失一副必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：补名只暂缓一日，王兆陪掌握黑袖套线索
  - 必做事件：
    1. 询问洗笔房杂役所见袖套细节
    2. 查值房和库房袖套
    3. 发现旧架上少一副
  - 禁止越界：
    1. 不把黑袖套直接等同瘦书办
    2. 不撤掉王某疑名
    3. 不公开真残墨
  - 结束状态：黑袖套从模糊特征变成夜值共用物
  - 下章承接：追查谁夜里领过纸
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 洗笔房追问，负责承接
    2. 值房查袖套，负责具体化
    3. 库房对照缺失，负责钩子
  - 字数支撑理由：线索从一人所见扩展到共用物，有现场对照和时间压力
  - 场景级字数预算：
    1. 场景1：约600~750字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：中
  - 流程章落地检查：黑袖套落到值房和库房实物
  - 最低可撑体量判断：能。

## 第182章：代领的人
- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：灰墨水痕指向夜里领纸的人，但那人只是替别人拿纸
- **本章兑现点**：灰墨水痕与领纸现场挂钩，但不直接结案
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪查纸坊补领记录回府对人，锁定一个抄房夜值小吏；小吏手上确有灰墨水痕，却说自己只是按牌代领旧纸，牌子来自值房夹板
- **冲突类型**：追人 / 代领断点
- **人物发展**：主角从查物转为查夜值牌流转
- **信息释放**：灰墨水痕说明领纸人接触过洗字水或湿墨，但代领关系让真执行人仍藏着
- **钩子**：小吏说交纸时看见袖套缺了一根系绳
- **本章不可删场景**：找到夜值小吏、手上灰墨水痕、代领牌必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：值房少一副黑袖套，主角要查夜里谁领纸
  - 必做事件：
    1. 找到夜值小吏
    2. 发现灰墨水痕
    3. 小吏交代只是代领
  - 禁止越界：
    1. 不让小吏知道全局
    2. 不让小吏直接指认瘦书办
    3. 不解除补名压力
  - 结束状态：线索转向值房夹板和缺系绳袖套
  - 下章承接：曹书吏压缩限期，逼午后补名
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 查夜值小吏，负责追人
    2. 灰墨水痕，负责线索关联
    3. 代领牌和缺系绳，负责转折
  - 字数支撑理由：追人、盘问和代领断点能形成推进
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~900字
    3. 场景3：约600~750字
  - 摘要风险判断：低
  - 流程章落地检查：灰墨水痕落到领纸人和代领牌
  - 最低可撑体量判断：能。

## 第183章：午后补名
- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：曹书吏不等明日，直接把补名时间压到午后
- **本章兑现点**：一日限期具体推进，不允许主角慢查
- **最晚冲突落点**：前20%
- **核心事件**：曹书吏以“原字不可复原”为由催办，要求午后将王某疑名补全；王兆陪只能拿代领牌和缺袖套争取查值房夹板的半日
- **冲突类型**：限期压缩 / 会场对抗
- **人物发展**：主角在更短时间内选择先查使用记录，不再追抽象线索
- **信息释放**：对手知道纸料证据边界，开始用边界反压主角
- **钩子**：值房夹板被人提前取走
- **本章不可删场景**：午后补名压力和夹板被取走必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角发现代领牌来自值房夹板
  - 必做事件：
    1. 曹书吏压缩补名到午后
    2. 主角用代领牌和袖套缺绳争半日
    3. 值房夹板被人取走
  - 禁止越界：
    1. 不撤王某疑名
    2. 不让曹书吏退场
    3. 不让主角无代价获得完整记录
  - 结束状态：补名时间逼近，值房夹板缺失
  - 下章承接：查袖套残线和共用记录
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 补名会压缩限期，负责压力
    2. 半证争取查值房，负责对抗
    3. 夹板被取走，负责损失
  - 字数支撑理由：限期、会场和证据损失能撑开
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约800~950字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：限期和记录缺失产生人物后果
  - 最低可撑体量判断：能。

## 第184章：残线
- **功能**：查 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪找到黑袖套残线，却发现袖套被多人共用
- **本章兑现点**：黑袖套不空转，但也不能直接定人
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪在值房旧架和库房挂钩下找到一截黑色系绳残线；残线能对上缺袖套，但值房老吏说夜里抄房、封袋处、库房都借过这副袖套
- **冲突类型**：实物推进 / 共用稀释
- **人物发展**：主角从找袖套所有者转为找夜间借用顺序
- **信息释放**：黑袖套证明夜里有人跨房口活动，但不能锁定单人
- **钩子**：老吏记得最后归还袖套的人袖口有湿灰水痕
- **本章不可删场景**：残线实物、共用记录、湿灰水痕必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：值房夹板被取走，补名午后逼近
  - 必做事件：
    1. 找到黑色系绳残线
    2. 证明它能对上缺袖套
    3. 得知袖套多人共用
  - 禁止越界：
    1. 不直接证明谁洗字
    2. 不让老吏完整记得全部人
    3. 不让补名压力消失
  - 结束状态：黑袖套线索具体化，但转向借用顺序
  - 下章承接：确认有人夜里进过洗笔房
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 值房旧架找残线，负责动作
    2. 对照缺袖套，负责证据推进
    3. 老吏说明共用，负责边界
  - 字数支撑理由：实物发现、对照和共用反转能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约700~850字
    3. 场景3：约650~800字
  - 摘要风险判断：低
  - 流程章落地检查：黑袖套落到残线和共用记录
  - 最低可撑体量判断：能。

## 第185章：夜里进门
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：黑袖套只能证明有人夜里进过洗笔房，仍不能证明谁洗字
- **本章兑现点**：黑袖套线索给出阶段结论和下一轮方向
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪把代领牌、残线、湿灰水痕和杂役证词对齐，只能证明夜里有人穿黑袖套进过洗笔房；曹书吏抓住“不能定人”继续午后补名，王兆陪转向查夜值牌和门禁记录
- **冲突类型**：半证边界 / 新追查方向
- **人物发展**：主角继续接受半证推进，不急于终局翻盘
- **信息释放**：下一批要查夜值牌、门禁记录和最后归还袖套的人
- **钩子**：门禁册上夜半有一处空押，正好在洗字水被倒前
- **本章不可删场景**：证据对齐、不能定人、门禁空押必须落地
- **视角**：王兆陪
- **节奏**：双峰

  - 必做事件：
    1. 对齐代领牌、残线、湿灰水痕
    2. 得出“有人夜进洗笔房”的有限结论
    3. 发现门禁册空押
  - 禁止越界：
    1. 不证明洗字人身份
    2. 不洗清王兆陪
    3. 不让曹书吏停止补名
  - 结束状态：线索转向夜值牌和门禁记录
  - 下章承接：追查门禁空押和最后归还袖套的人
- **章节可写性证明**：

  - 场景拆分：

    2. 曹书吏反压，负责人物后果
    3. 门禁空押，负责新钩子
  - 字数支撑理由：证据边界、补名压力和门禁新线足够支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
