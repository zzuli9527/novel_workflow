# 第186~190章批次评审（v1）

## 一、总体判断

第186~190章通过了“文书阻断去重检查”的回归验证。本批没有继续使用“册子被调走 / 记录被取走 / 账本被封存 / 夹板被拿走”作为主要阻断，而是把门禁空押转成了值门人、夜值牌、牌面刮痕、人物受罚、当众验牌和口音追查。

这一批的关键验证点成立：

1. 门禁空押牵出了具体值门人赵平
2. 夜值牌和门禁记录都没有丢
3. 阻断方式从“记录缺失”改成“牌面被刮 + 值门人被罚 + 证据边界被反压”
4. 午后补名压力持续推动行动
5. 至少一个现场人物赵平承担后果
6. 门禁空押没有直接证明洗字人，只转向抄房夜值和声音误导

## 三、本批主要问题

### 1. 新增规则有效

`03-chapters.md` 新增的文书阻断去重规则在本批生效：

- 没有再让关键记录被拿走
- 门禁册和夜值牌都保留在现场
- 阻力来自人不敢写名、牌被刮、赵平被罚、证据边界被曹书吏反压
- 每次阻断都带来新追查方向

### 2. 人物后果比文书阻断更有效

赵平被罚让门禁线更有代价，也避免了“查一本册子又发现没了”的重复感。这个方向应继续保留：流程章不只查物，还要让经手人承担压力。

### 3. 下一批要避免“口音线索悬浮”

口音线索如果继续往下写，容易变成模糊判断。下一批必须让陈录、孙庆进入现场动作：

- 两人分别在抄房夜值中承担什么职责
- 谁接触过曹书吏文书
- 谁能拿到夜值牌
- 谁有模仿口音或压低声音的动机
