# 新模具复跑细纲 v1：第186~190章

> 用途：回归验证“文书阻断去重检查”，确认门禁空押线不再靠“记录被拿走”空转，而是落到值门人、门房动作、夜值牌实物和人物后果。

## 批次目标

- 验证门禁空押牵出具体值门人
- 验证夜值牌和门禁记录不同时缺失
- 验证午后补名压力持续推动行动
- 验证至少一个现场人物承担后果
- 验证门禁空押只能推进追查，不能直接证明洗字人

## 批次风险点

- 继续查册而不查人
- 值门人直接说出真名导致过快收束
- 夜值牌变成终局铁证
- 午后补名只停留在口头压力

---

## 第186章：值门人
- **功能**：查 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪从门禁空押追到当夜值门人，对方承认有人持夜值牌进门
- **本章兑现点**：门禁空押落到具体人物和开门动作
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪到门房找到当夜值门人赵平，赵平承认夜半有人持牌进洗笔房，但对方压低声音、戴黑袖套，他不敢写名
- **冲突类型**：现场盘问 / 值门人自保
- **人物发展**：主角从查门禁册转为查门房人物
- **信息释放**：空押不是无缘无故留白，而是值门人怕担责故意不写
- **钩子**：赵平说夜值牌没有丢，还压在门房牌匣里
- **本章不可删场景**：门房盘问、开门回忆、夜值牌未丢必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角掌握门禁空押和黑线毛
  - 必做事件：
    1. 找到当夜值门人赵平
    2. 赵平承认有人持牌进门
    3. 确认夜值牌还在门房
  - 禁止越界：
    1. 不让赵平说出真名
    2. 不让门禁空押直接证明洗字人
    3. 不撤王某疑名
  - 结束状态：空押牵出值门人和夜值牌
  - 下章承接：当场验夜值牌
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 门房找赵平，负责查人
    2. 夜半开门回忆，负责现场动作
    3. 夜值牌未丢，负责钩子
  - 字数支撑理由：门房盘问、赵平自保和夜值牌实物能支撑展开
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：门禁空押绑定值门人和开门动作
  - 最低可撑体量判断：能。

## 第187章：牌面刮痕
- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：夜值牌还在，但牌面被刮过，赵平当场被曹书吏责罚
- **本章兑现点**：不使用“记录被拿走”，改用实物损伤和人物后果推进
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪验牌，发现牌面签痕被刮浅；曹书吏借机责罚赵平擅留空押，试图把问题压成值门人失职
- **冲突类型**：验牌现场 / 推责压人
- **人物发展**：主角意识到对手想让值门人背锅
- **信息释放**：牌还在，记录也在，但牌面被动过
- **钩子**：赵平被罚时低声说牌缝里还有东西
- **本章不可删场景**：验牌、牌面刮痕、赵平被罚必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：夜值牌还在门房
  - 必做事件：
    1. 当场验夜值牌
    2. 发现牌面刮痕
    3. 赵平因空押被责罚
  - 禁止越界：
    1. 不让夜值牌直接定人
    2. 不让赵平完整招供
    3. 不让曹书吏退场
  - 结束状态：赵平受罚，牌缝成为新线索
  - 下章承接：午后补名临近，主角争取当众验牌缝
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 门房验牌，负责实物推进
    2. 曹书吏责罚，负责人物后果
    3. 牌缝提示，负责钩子
  - 字数支撑理由：验牌、推责和受罚能形成现场张力
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：记录未丢，阻断方式改为牌面被刮和人受罚
  - 最低可撑体量判断：能。

## 第188章：当众验牌
- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：午后补名临近，王兆陪用赵平被罚证明有人急着灭口，逼府里当众验牌
- **本章兑现点**：午后压力推动主角争取公开验牌机会
- **最晚冲突落点**：前20%
- **核心事件**：曹书吏要继续午后补名，王兆陪指出赵平被罚会让门禁空押变成一人失职，要求在补名前当众验牌缝
- **冲突类型**：会场对抗 / 争取验牌
- **人物发展**：主角从私下查证转为公开争取程序
- **信息释放**：对手越急着罚赵平，越说明门禁空押不是普通失误
- **钩子**：牌缝里露出一根黑色残线
- **本章不可删场景**：午后补名会、赵平受罚反用、当众验牌必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：赵平被罚，牌缝有疑点
  - 必做事件：
    1. 曹书吏推动午后补名
    2. 主角用赵平受罚争取验牌
    3. 当众发现黑色残线
  - 禁止越界：
    1. 不洗清主角
    2. 不让赵平直接翻供
    3. 不让验牌成为终局
  - 结束状态：黑线从门禁册旁延伸到夜值牌缝
  - 下章承接：验证残线和黑袖套缺绳
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 午后补名会，负责压力
    2. 争取当众验牌，负责对抗
    3. 牌缝黑线，负责推进
  - 字数支撑理由：限期、程序争夺和当众验牌能支撑展开
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：午后补名推动现场验牌
  - 最低可撑体量判断：能。

## 第189章：牌缝黑线
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：牌缝黑线能对上缺绳黑袖套，却只能证明持牌人穿过那副袖套
- **本章兑现点**：夜值牌形成半证，不成为铁证
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪把牌缝黑线与前一批残线对照，确认同类断口；但袖套多人共用，夜值牌只能证明持牌人穿过缺绳袖套
- **冲突类型**：证据对照 / 证据边界
- **人物发展**：主角继续接受半证边界
- **信息释放**：持牌进门者和缺绳袖套存在关联
- **钩子**：赵平听到验线后说那人开口像抄房口音
- **本章不可删场景**：对照黑线、证据边界、抄房口音必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：夜值牌缝出现黑线
  - 必做事件：
    1. 对照黑线和缺绳袖套
    2. 证明持牌人与袖套有关
    3. 明确不能证明洗字人
  - 禁止越界：
    1. 不让夜值牌直接定人
    2. 不让袖套线终结
    3. 不撤补名压力
  - 结束状态：半证成立，线索转向抄房口音
  - 下章承接：赵平回忆声音但不敢确认
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 残线对照，负责小兑现
    2. 曹书吏反压证据边界，负责压力
    3. 赵平说出口音，负责新方向
  - 字数支撑理由：实物对照、会场反压和口音线索能支撑展开
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约750~900字
    3. 场景3：约550~700字
  - 摘要风险判断：中
  - 流程章落地检查：夜值牌不是终局铁证，只给出追查方向
  - 最低可撑体量判断：能。

## 第190章：像抄房的人
