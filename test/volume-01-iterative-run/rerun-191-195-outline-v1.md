# 新模具复跑细纲 v1：第191~195章

> 用途：验证口音线索能否落到陈录、孙庆的现场动作，而不是停留在“像不像”的听感推理。

## 批次目标

- 验证陈录、孙庆都进入具体现场动作
- 验证口音线索绑定喊门、送文、交牌、压嗓等动作
- 验证曹书吏与陈录送文关联只形成疑点，不能直接定罪
- 验证午后补名压力继续存在
- 验证王某疑名继续保留

## 批次风险点

- 赵平直接认人，导致线索过快收束
- 陈录因送文关联被直接定罪
- 孙庆的模仿能力变成便利答案
- 口音线索变成纯听感推理

## 第191章：两个北巷口音
- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪在抄房见到陈录、孙庆，两人口音相近，但夜值职责不同
- **本章兑现点**：陈录、孙庆从名单变成现场人物
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪到抄房找陈录、孙庆，发现陈录负责夜里送文，孙庆负责搬旧纸和守后案，两人都能说出类似赵平听到的北巷口音
- **冲突类型**：现场查人 / 双目标分流
- **人物发展**：主角从查名单转为查具体夜值动作
- **信息释放**：两人都有可能接触夜值牌或洗笔房路线，但职责不同
- **钩子**：陈录袖口有新纸灰，孙庆手背有旧绳勒痕
- **本章不可删场景**：抄房见两人、职责区分、纸灰与勒痕必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角掌握陈录、孙庆两个同口音方向
  - 必做事件：
    1. 到抄房见陈录、孙庆
    2. 区分两人夜值职责
    3. 发现陈录纸灰、孙庆绳痕
  - 禁止越界：
    1. 不让赵平直接认人
    2. 不让陈录或孙庆招供
    3. 不撤王某疑名
  - 结束状态：两人都成为可查现场人物
  - 下章承接：查陈录替曹书吏送文
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 抄房找人，负责从名单落地
    2. 两人职责对照，负责分流
    3. 纸灰与勒痕，负责钩子
  - 字数支撑理由：查人、对话、职责对照和手部细节能形成现场推进
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：口音线索绑定抄房现场和人物职责
  - 最低可撑体量判断：能。

## 第192章：只是送文
- **功能**：查 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：陈录承认替曹书吏送过文，却坚持自己没进洗笔房
- **本章兑现点**：曹书吏与陈录关联成为疑点，但不直接定罪
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪跟陈录复走送文路线，确认他确实替曹书吏送过一封急件，但路线只到抄房侧门，未必进洗笔房
- **冲突类型**：路线复查 / 疑点边界
- **人物发展**：主角从“关联可疑”转为“关联不足以定罪”
- **信息释放**：陈录有接触曹书吏文书的机会，但没有直接洗字机会
- **钩子**：陈录说送文时孙庆正从门房方向抱纸回来
- **本章不可删场景**：送文路线复走、曹书吏急件、孙庆抱纸必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：陈录有纸灰且曾替曹书吏送文
  - 必做事件：
    1. 陈录承认送文
    2. 复走送文路线
    3. 得知孙庆从门房方向抱纸回来
  - 禁止越界：
    1. 不因送文直接定陈录
    2. 不让陈录承认洗字
    3. 不让曹书吏失去压迫力
  - 结束状态：陈录线形成疑点边界，孙庆线加重
  - 下章承接：查孙庆压嗓和抱纸
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 盘问陈录，负责疑点
    2. 复走路线，负责现场动作
    3. 孙庆抱纸，负责转向
  - 字数支撑理由：路线复查和陈录自保能支撑冲突
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：送文关联落到路线复走，不是直接推理定罪
  - 最低可撑体量判断：能。

## 第193章：压着嗓子
- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：孙庆能压低声音学陈录说话，却把它说成夜值玩笑
- **本章兑现点**：模仿口音成为可疑能力，但不能成为便利答案
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪在抄房后案逼孙庆复述夜里喊门的话，孙庆能压嗓学陈录口音，却说夜里互相学话常有；他手背绳痕来自搬纸捆
- **冲突类型**：现场试声 / 自保解释
- **人物发展**：主角发现“声音可伪装”，不能只靠赵平听感
- **信息释放**：孙庆具备制造声音误导的条件，但仍缺少持牌证据
- **钩子**：孙庆搬纸捆的绳痕与黑袖套缺绳粗细相近
- **本章不可删场景**：后案试声、孙庆压嗓、绳痕对照必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：陈录说孙庆曾从门房方向抱纸回来
  - 必做事件：
    1. 逼孙庆现场压嗓
    2. 孙庆解释是夜值玩笑
    3. 对照手背绳痕
  - 禁止越界：
    1. 不让孙庆承认伪装
    2. 不让声音线索直接定人
    3. 不让口音替代物证
  - 结束状态：声音可伪装，线索转向绳痕和牌缝木屑
  - 下章承接：午后补名再压，主角用声音可伪装拖住认人
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 抄房后案试声，负责现场动作
    2. 孙庆解释夜值玩笑，负责人物自保
    3. 绳痕对照，负责新线索
  - 字数支撑理由：试声、对抗和绳痕细节能撑开
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约750~900字
    3. 场景3：约600~750字
  - 摘要风险判断：低
  - 流程章落地检查：口音线索绑定试声动作和手部痕迹
  - 最低可撑体量判断：能。
