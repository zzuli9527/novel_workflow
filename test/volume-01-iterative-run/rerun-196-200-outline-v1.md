# 新模具复跑细纲 v1：第196~200章

> 用途：验证木屑线索能否进入牌匣、手指、袖口、洗手盆、纸灰等现场动作，避免退成技术比对说明。

## 批次目标

- 验证木屑线索不技术化
- 验证陈录、孙庆都保留疑点但不直接招供
- 验证至少出现一次清理痕迹动作
- 验证午后补名压力继续推动限时查证
- 验证木屑只能证明接触夜值牌，不能证明洗字人

## 批次风险点

- 木屑写成鉴定报告
- 洗手或换袖直接坐实真凶
- 陈录、孙庆其中一人过早出局
- 午后补名压力消失

---

## 第196章：牌匣缺口
- **功能**：查 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪查牌匣木刺，发现牌匣边缘新缺口能刮出同类木屑
- **本章兑现点**：木屑线索进入牌匣现场
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪回门房查牌匣，发现牌匣底边有新缺口，夜值牌抽出时会刮出细木屑；赵平承认昨夜有人抽牌很急，曾把牌磕在匣边
- **冲突类型**：现场验物 / 开门回忆
- **人物发展**：主角从查身上木屑转为查木屑产生现场
- **信息释放**：木屑来源不是抽象物证，而是门房牌匣动作
- **钩子**：牌匣缺口旁还蹭着一点湿灰，像有人湿手抽过牌
- **本章不可删场景**：门房查牌匣、抽牌刮木屑、湿灰痕必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角掌握陈录袖口木粉、孙庆指缝木屑
  - 必做事件：
    1. 查牌匣缺口
    2. 验证抽牌会刮出木屑
    3. 发现湿灰痕
  - 禁止越界：
    1. 不让木屑直接定人
    2. 不让赵平确认真名
    3. 不撤王某疑名
  - 结束状态：木屑来源落到牌匣动作
  - 下章承接：查谁湿手或洗手
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 门房查牌匣，负责现场动作
    2. 抽牌刮木屑，负责物证来源
    3. 湿灰痕，负责钩子
  - 字数支撑理由：查物、试抽、赵平回忆和湿灰痕能支撑展开
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：木屑落到牌匣和抽牌动作
  - 最低可撑体量判断：能。

## 第197章：洗手盆底
- **功能**：查 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：孙庆被发现洗手，指缝木屑少了，但洗手盆底留下木粉
- **本章兑现点**：清理痕迹动作出现，但不直接坐实孙庆
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪追到后院水盆，发现孙庆刚洗过手，指缝木屑被洗掉一半；盆底有细木粉和纸绳灰，孙庆解释是搬纸后洗手
- **冲突类型**：现场撞见 / 自保解释
- **人物发展**：主角确认对手可能在处理痕迹
- **信息释放**：孙庆既可能处理牌匣木屑，也可能只是洗搬纸痕迹
- **钩子**：水盆旁有一截新断纸绳，粗细接近黑袖套缺绳
- **本章不可删场景**：后院撞见洗手、盆底木粉、纸绳必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：牌匣缺口旁有湿灰痕
  - 必做事件：
    1. 撞见孙庆洗手
    2. 发现盆底木粉
    3. 孙庆用搬纸解释
  - 禁止越界：
    1. 不让孙庆招供
    2. 不让洗手直接定罪
    3. 不让陈录线出局
  - 结束状态：孙庆疑点加重但仍有解释
  - 下章承接：查陈录袖口木粉为何消失
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 后院追人，负责动作
    2. 洗手盆底，负责线索
    3. 孙庆解释，负责边界
  - 字数支撑理由：撞见、盘问和痕迹处理能形成现场张力
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：木屑落到洗手动作和盆底残留
  - 最低可撑体量判断：能。

## 第198章：换下的袖
- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：陈录换了袖口，旧袖带着纸灰木粉，被藏到抄房炭盆旁
- **本章兑现点**：陈录线继续保留，不被孙庆线替代
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪回抄房找陈录，发现陈录换了外袖；旧袖被压在炭盆旁，袖口纸灰里混着木粉，陈录说是怕纸灰弄脏送文
- **冲突类型**：现场搜找 / 换袖自保
- **人物发展**：主角确认陈录也在处理痕迹
- **信息释放**：陈录可能接触过牌匣，也可能只是被纸灰带上木粉
- **钩子**：旧袖内侧有一点门房湿灰，与牌匣缺口湿灰相近
- **本章不可删场景**：抄房找陈录、旧袖、炭盆旁纸灰木粉必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：孙庆洗手疑点加重，陈录袖口木粉仍未解释
  - 必做事件：
    1. 发现陈录换袖
    2. 找到旧袖
    3. 旧袖上有纸灰木粉和湿灰
  - 禁止越界：
    1. 不让陈录招供
    2. 不让旧袖直接定罪
    3. 不让孙庆线消失
  - 结束状态：陈录、孙庆都处理过痕迹
  - 下章承接：午后补名再压，用两人都处理痕迹拖住草率补名
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 抄房找陈录，负责追人
    2. 炭盆旁找旧袖，负责现场动作
    3. 纸灰木粉和湿灰，负责线索
  - 字数支撑理由：追问、搜找和换袖解释能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：木屑落到换袖、藏袖和炭盆现场
  - 最低可撑体量判断：能。

- **节奏**：双峰

## 第200章：只能证明碰过牌
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：木屑只能证明陈录、孙庆都接触过夜值牌，不能证明谁持牌进门
- **本章兑现点**：物证线索形成阶段结论和下一轮方向
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪整理牌匣缺口、孙庆洗手、陈录换袖和两道脚步声，确认两人都可能接触过夜值牌，但仍无法证明谁持牌进门；下一步必须查两人昨夜先后时间顺序
- **冲突类型**：证据边界 / 时间顺序转向
- **人物发展**：主角继续坚持半证推进，不草率定人
- **信息释放**：可能存在一人拿牌、一人遮掩或嫁祸的配合
- **钩子**：门房外泥地两道脚印深浅不同，一前一后，不像同一时间离开
- **本章不可删场景**：证据整理、不能定人、脚印时间差必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：两人都处理过木屑痕迹，赵平补出两道脚步声
  - 必做事件：
    1. 整理牌匣、洗手、换袖证据
    2. 明确只能证明接触夜值牌
    3. 发现脚印时间差
  - 禁止越界：
    1. 不证明洗字人
    2. 不证明谁持牌进门
    3. 不撤补名压力
  - 结束状态：下一批转向陈录、孙庆的拿牌时间顺序
  - 下章承接：查门房外脚印和两人昨夜动线
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 半证整理，负责阶段收束
    2. 证据边界，负责压力
    3. 脚印时间差，负责新方向
  - 字数支撑理由：证据收束、反压和新现场线索能支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约700~850字
    3. 场景3：约600~750字
  - 摘要风险判断：中
  - 流程章落地检查：木屑不作终局鉴定，转向脚印和时间顺序
  - 最低可撑体量判断：能。
