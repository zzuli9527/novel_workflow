# 第201~205章批次评审（v1）

## 一、总体判断

第201~205章通过了“时间顺序不能复盘化”的验证。本批没有让王兆陪站在原地复述时间线，而是通过陈录路线重走、孙庆负重重走、公开当场重走、步幅差异、脚印深浅和“给我”低语推进。

这一批的关键验证点成立：

1. 时间顺序进入了现场重走
2. 陈录路线和孙庆路线都被实际复现
3. 脚印没有直接定人，只形成半证
4. 午后补名压力继续推动公开验证
5. 陈录、孙庆都没有招供
6. 线索转向“谁把夜值牌交给谁”

## 二、单章结果

- 第201章：陈录送文路线重走成立，浅脚印半证成立
- 第202章：孙庆搬纸路线重走成立，深脚印和门槛拖痕成立
- 第203章：补名压力逼出公开当场重走
- 第204章：两人重走都不完整，赵平补出“给我”
- 第205章：时间顺序阶段收束，转向交牌动作

## 三、本批主要问题

### 1. 时间顺序现场化有效

本批没有写成复盘说明，而是拆成：

- 陈录从曹书吏值房到抄房侧门的路线
- 侧门泥地和浅脚印
- 孙庆从后案抱纸到门房的路线
- 负重深脚印和门槛拖痕
- 当场公开重走
- 步幅差异和“给我”低语

这些都能形成正文章节的现场动作。

### 2. 半证边界稳定

时间顺序只完成了有限证明：

- 陈录能接近门房
- 孙庆路线更贴近深脚印
- 两人可能先后到过门房
- 夜值牌可能发生交接

它没有证明：

- 谁洗字
- 谁最终持牌进门
- 谁陷害王兆陪

这说明证据公开代价检查仍然有效。

### 3. 下一批要避免“交牌动作抽象化”

交牌动作如果继续往下写，容易变成“推测谁给了谁”。下一批必须落到：

- 门房墙缝纸灰
- 递牌时的手位、袖口、纸捆遮挡
- 谁的袖口或纸捆能擦到墙缝
- 谁因交牌追查被迫改口或改变动线说法
