# 新模具复跑正文：第201~205章

> 用途：验证时间顺序线索是否能进入现场重走，而不是写成事后复盘说明。

## 第201章 浅脚印

陈录不愿意重走。

“我已经说过了，送文只到抄房侧门。”

王兆陪没有和他争，直接让他从曹书吏值房门口开始走。

廊下很窄，转过封袋处外墙，再到抄房侧门。陈录走得熟，几乎不用看路。

王兆陪在侧门停住。

“如果从这里去门房，怎么走？”

陈录脸色一变：“我没去过。”

“走一遍。”

侧门到门房有一段泥地，平日被屋檐挡着，不算深。陈录踩上去，留下的脚印浅，间距也轻。

门房外昨夜那道浅痕，确实能由这条路留下。

可这只能证明一件事。

陈录有路能靠近门房。

不能证明他进了门。

王兆陪低头时，泥地旁还有一片暗红旧封泥碎，像从文袋上蹭落。

陈录也看见了。

他的喉结动了一下。

送文路线，比他说的更接近门房。

## 第202章 深脚印

孙庆抱起纸捆时，肩膀明显沉了一下。

王兆陪让他从抄房后案开始走。

后案到门房的路比陈录那条绕，经过一段木架，再穿过水缸旁的小道。孙庆抱着纸，右手压着纸捆，左手垂在下面。

如果左手拿着夜值牌，确实会被纸捆遮住。

孙庆像看出他在想什么，立刻道：“我只是搬纸。”

王兆陪没有答。

他看地。

孙庆负重走到门房外，泥地被踩得更深。那道深痕，与昨夜门外的深脚印更近。

孙庆把纸捆放下时，纸绳扫过门槛，拖出一道细痕。

赵平低声道：“昨夜门槛上，也有这么一道。”

孙庆脸色难看：“搬纸都会有。”

这话也说得通。

深脚印不能证明他拿牌。

但能证明，他的路线和昨夜痕迹贴得更紧。

## 第203章 当场重走

曹书吏不想再听脚印。

他把疑案急请副本摊开：“一个浅印，一个深印，能说明什么？府里来往的人多，王某书生不能一直空着。”

王兆陪看着那张副本。

“那就当场重走。”

曹书吏抬眼：“刚才不是走过？”

“刚才是各走各的。”王兆陪道，“现在在门房外，当着所有人，按昨夜门外痕迹重走。”

屋里几个书办互相看了一眼。

这不是推理。

这是让脚印、步幅、负重、门槛拖痕在众人眼前重现。

曹书吏不能直接拒。

陈录听见“当场重走”四个字，下意识看了孙庆一眼。

那一眼很短。

但王兆陪看见了。

如果两人昨夜互不相干，陈录为什么要先看孙庆？

曹书吏沉着脸：“走。走完若仍不能证明，午后补名照旧。”

王兆陪点头。

他要的就是这个当场。

## 第204章 步子对不上

陈录先走。

他从侧门方向过来，踩上浅泥地，步子轻，脚印也浅。

可间距对不上。

昨夜浅脚印之间更开一些，像走的人急，步子放得大。陈录当场走出来的步子短了半寸。

陈录立刻道：“夜里急，步子自然不同。”

王兆陪没有反驳。

孙庆接着抱纸走。

他负重之后，脚印深，门槛拖痕也像。可他走到门房外时，停的位置比昨夜深痕偏后。

也不完全对。

两个人都像。

也都不完整。

赵平站在门边，忽然开口：“昨夜后来的那人停下前，我听见前头有人低声说了一句。”

“什么？”

赵平咽了咽：“给我。”

给我。

不是一个人独自拿牌进门。

更像一个人先到门边，一个人后来接过什么。

## 第205章 谁把牌给谁

脚印不能定人。

王兆陪把浅脚印、深脚印、门槛拖痕和赵平听见的“给我”摆在一起。

它们能证明两个人先后到过门房。

一个从侧门方向来，像陈录。

一个负重靠近，像孙庆。

可谁先进门，谁拿牌，谁把牌交给谁，仍然没有答案。

曹书吏立刻抓住这一点：“既然没有答案，补名不该再拖。”

“没有答案，才不能补。”王兆陪道，“若有人先取牌，再交给后来的人进门，那门禁空押就不是一个人的事。”

他回到门房牌匣旁。

墙缝里卡着一点纸灰，很细，像递牌时袖口擦过墙面留下的。

纸灰不是脚印。

却说明有人在牌匣旁有过递接动作。

王兆陪看向陈录，又看向孙庆。

时间顺序只能说明他们都来过。

下一步，要查的是那块夜值牌，到底从谁手里交到了谁手里。
