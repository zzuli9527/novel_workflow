# 第206~210章批次评审（v1）

## 一、总体判断

第206~210章通过了“交牌动作不能抽象化”的验证。本批没有用一句“谁把牌给谁”替代推理，而是通过墙缝纸灰、侧身动作、陈录递孙庆、孙庆递陈录、纸捆遮挡、门槛拖痕和门外剩人完成现场模拟。

这一批的关键验证点成立：

1. 交牌动作进入现场模拟
2. 两种交牌方向都被实际模拟
3. 交牌没有直接证明洗字人
4. 陈录、孙庆都没有招供
5. 午后补名压力继续存在
6. 线索转向谁进洗笔房、谁留在门外

## 二、单章结果

- 第206章：墙缝纸灰和侧身递牌成立
- 第207章：陈录递孙庆方向可行但不完整
- 第208章：孙庆递陈录方向也可行但不完整
- 第209章：两种模拟反压单人背锅，赵平补出门外剩人
- 第210章：交牌阶段收束，转向进门/留门外分工

## 三、本批主要问题

### 1. 交牌动作现场化有效

本批把交牌拆成了具体动作：

- 正常抽牌不会擦墙缝
- 侧身递牌会擦墙缝
- 陈录袖口可解释纸灰
- 孙庆纸捆可遮挡赵平视线
- 反向模拟也能解释部分痕迹
- 交牌后门外仍有一人

这些都能支撑正文现场，而不是推理摘要。

### 2. 半证边界稳定

交牌只完成了有限证明：

- 夜值牌可能转手
- 陈录、孙庆都可能参与转手
- 赵平看不清有合理现场原因
- 进门人与门外剩人可能不是同一个人

它没有证明：

- 谁最终进洗笔房
- 谁洗字
- 谁陷害王兆陪

### 3. 下一批要避免“分工线抽象化”

进门/留门外分工如果继续往下写，容易变成“一个望风一个动手”的抽象判断。下一批必须落到：

- 洗笔房门槛内外痕迹
- 封袋处退步痕
- 谁能从封袋处看见洗笔房
- 谁留在门外时做了具体动作
- 谁进门后留下洗字或水盆痕迹
