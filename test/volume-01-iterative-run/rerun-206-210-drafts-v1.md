# 新模具复跑正文：第206~210章

> 用途：验证交牌动作是否能进入现场模拟，而不是主角口头推测谁把牌给谁。

## 第206章 墙缝纸灰

墙缝里的纸灰很少。

少到如果不蹲下看，根本看不见。

王兆陪没有急着说是谁留下的。他先站到牌匣前，按正常取牌的姿势伸手。

手碰不到墙。

袖口也碰不到。

他又往旁边挪了半步，侧身，像给身后人让位置，再把夜值牌往身后递。

这一下，袖口擦到了墙缝。

纸灰的位置正对。

赵平站在门边，看着他的动作，脸色变了。

“想起来了？”

赵平点头：“昨夜门边有人挪过一步。不是走，是侧着让了一下。”

“让谁？”

赵平摇头。

他仍然不敢说，也未必看见。

但墙缝纸灰不是凭空来的。

正常抽牌不会蹭到那里。

只有侧身递牌，或者侧身接牌，才会把袖口、纸捆或手背擦过去。

交牌这件事，终于从一句猜测，落成了一个动作。

## 第207章 陈录递牌

陈录先站到浅脚印的位置。

他脸色难看，却还是按王兆陪说的，伸手从牌匣里抽出夜值牌。

“侧身。”

陈录侧过身，袖口擦到墙缝。

纸灰能解释。

他把牌往后递。

孙庆抱着纸捆站在他身后，纸捆挡住了赵平从门内看出来的视线。若昨夜真是这样，赵平只能看见黑袖套和纸捆影子，未必能看清牌从谁手里到谁手里。

孙庆接牌时，纸绳扫过门槛。

拖痕也能解释。

可动作不顺。

陈录递牌时，手要绕过纸捆，孙庆接牌要退半步。昨夜深脚印的位置，比这个更靠前。

陈录立刻道：“你看，不对。”

王兆陪没有否认。

这条路可行。

但不完整。

## 第208章 孙庆递牌

第二次，孙庆先站到牌匣旁。

他抱着纸捆，身体往门边一挤，赵平从门内看出来，视线几乎全被纸捆挡住。

王兆陪让他伸手取牌。

孙庆的手藏在纸捆后面。

如果夜里光暗，确实看不见。

他把牌递给陈录。

陈录从侧门方向伸手接牌，袖口在墙缝上轻轻一擦，纸灰也能落进去。

陈录接牌后退回浅泥地，留下浅脚印方向。

这一次，浅脚印更顺。

但问题也在这里。

如果陈录接牌后退走，那真正进洗笔房的人未必是他。

如果孙庆取牌后留在门外，那纸捆和深脚印也不能证明他进门。

两种模拟，都有一半对。

也都有一半缺。

王兆陪看着两人：“所以，不能只定一个。”

## 第209章 都不能排除

曹书吏最先抓孙庆。

“抱纸遮视线，深脚印又像他。王兆陪，你还说不能定？”

孙庆脸色一白。

陈录明显松了一口气。

王兆陪看见了。

他把两次模拟的结果摆出来：“若只看深脚印，像孙庆。若看墙缝纸灰和浅脚印，又绕不开陈录。两种动作都能解释一部分，谁都不能直接背完这件事。”

曹书吏冷声道：“那就都不能证明。”

“都不能证明，所以不能拿来补我的名字。”

这句话让屋里安静下来。

补名的笔又停住了。

赵平这时低声补了一句：“昨夜进门的人拿牌后，门外还有一个人没走。”

王兆陪转头。

赵平攥着门框：“我听见门关上后，外头还有一步，往旁边退了。”

一个进门。

一个留在门外。

交牌之后，真正该查的，是谁进了洗笔房，谁留在门外遮掩。

## 第210章 谁留在门外

交牌证明不了谁洗字。

它甚至证明不了谁最终拿牌进门。

王兆陪把墙缝纸灰、纸捆遮挡、浅脚印、深脚印和赵平说的门外剩人放在一起。

这些东西只说明，夜值牌很可能转过手。

有人递。

有人接。

然后一个人进了洗笔房，另一个人留在门外。

曹书吏立刻道：“仍然不能证明你不是王某书生。”

王兆陪没有否认。

他现在所有的证据，都只能拖住那支笔。

不能砸断那支笔。

他走到门房外，看地上那道退步痕。

那一步不是往抄房退。

也不是往后案退。

而是斜斜退向封袋处。

王兆陪停住。

如果有人进洗笔房洗字，另一个人为什么往封袋处退？

门外剩人，也许不是望风。

他可能在等封袋那边的人。
