# 新模具复跑细纲 v1：第206~210章

> 用途：验证交牌动作能否进入现场模拟，而不是写成主角推测“谁把牌给谁”。

## 批次目标

- 验证交牌动作落到墙缝、袖口、纸捆、手位、遮挡角度
- 验证至少两种交牌方向都被现场模拟
- 验证交牌只能证明夜值牌转手，不能证明洗字人
- 验证午后补名压力继续存在

## 批次风险点

- 交牌只停留在口头推测
- 模拟直接定罪
- 陈录、孙庆其中一人过早出局
- 补名压力消失

---

## 第206章：墙缝纸灰
- **功能**：查 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪查门房墙缝纸灰，发现只有靠近牌匣侧身递牌才会擦到墙缝
- **本章兑现点**：交牌线索进入墙缝和手位现场
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪在门房牌匣旁模拟取牌，发现正常抽牌不会碰到墙缝，只有侧身让位、把牌递给身后人时，袖口或纸捆才会擦到墙缝纸灰位置
- **冲突类型**：现场模拟 / 动作边界
- **人物发展**：主角从查时间顺序转为查交牌动作
- **信息释放**：墙缝纸灰不是普通蹭灰，而是递牌姿势留下的可能痕迹
- **钩子**：赵平说昨夜开门前，确实听见门边有人侧身挪了一步
- **本章不可删场景**：牌匣旁模拟取牌、侧身递牌、墙缝纸灰必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：主角掌握“给我”低语和墙缝纸灰
  - 必做事件：
    1. 模拟正常抽牌
    2. 模拟侧身递牌
    3. 赵平补出侧身挪步声
  - 禁止越界：
    1. 不证明谁交牌
    2. 不证明谁进洗笔房
    3. 不撤王某疑名
  - 结束状态：交牌动作进入现场模拟
  - 下章承接：模拟陈录递牌给孙庆
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 正常抽牌模拟，负责排除
    2. 侧身递牌模拟，负责线索
    3. 赵平补充听觉细节，负责钩子
  - 字数支撑理由：两种动作模拟、赵平回忆和墙缝痕迹能支撑展开
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：交牌动作落到牌匣、墙缝和侧身动作
  - 最低可撑体量判断：能。

## 第207章：陈录递牌
- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：模拟陈录递牌给孙庆，陈录袖口能擦到墙缝，但孙庆纸捆遮挡更贴近昨夜
- **本章兑现点**：第一种交牌方向被现场验证，但不能定案
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪让陈录按浅脚印位置取牌，再侧身递给抱纸的孙庆；陈录袖口能擦到墙缝，孙庆纸捆能遮住赵平视线，但动作略显别扭
- **冲突类型**：现场模拟 / 自保解释
- **人物发展**：主角确认陈录递牌给孙庆可行但不完整
- **信息释放**：陈录可作为先取牌者，孙庆可作为遮挡接牌者
- **钩子**：孙庆抱纸接牌时，纸绳会扫出门槛拖痕
- **本章不可删场景**：陈录取牌、侧身递牌、孙庆纸捆遮挡必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：侧身递牌能解释墙缝纸灰
  - 必做事件：
    1. 陈录按浅脚印位置取牌
    2. 陈录递牌给孙庆
    3. 孙庆纸捆遮挡和拖痕出现
  - 禁止越界：
    1. 不定陈录是先取牌者
    2. 不定孙庆进门
    3. 不让两人招供
  - 结束状态：陈录递孙庆方向可行但不完整
  - 下章承接：模拟孙庆递牌给陈录
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 陈录取牌，负责动作起点
    2. 递牌给孙庆，负责交接
    3. 纸捆遮挡和拖痕，负责半证
  - 字数支撑理由：动作复现、两人自保和赵平视线遮挡能支撑展开
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：交牌方向进入现场模拟
  - 最低可撑体量判断：能。

## 第208章：孙庆递牌
- **功能**：查 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：反向模拟孙庆递牌给陈录，纸捆能挡住赵平视线，陈录接牌后更容易留下浅脚印
- **本章兑现点**：第二种交牌方向成立，保留双疑点
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪让孙庆先抱纸靠近牌匣，借纸捆遮挡取牌，再把牌递给陈录；陈录接牌后从浅泥地撤走，更贴近浅脚印，但墙缝纸灰位置略偏
- **冲突类型**：反向模拟 / 证据边界
- **人物发展**：主角确认两种交牌方向都不能排除
- **信息释放**：孙庆也可能是先取牌者，陈录也可能是接牌者或遮掩者
- **钩子**：陈录接牌时袖口纸灰更容易落进墙缝
- **本章不可删场景**：孙庆取牌、纸捆遮挡、陈录接牌退走必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：陈录递孙庆方向可行但不完整
  - 必做事件：
    1. 孙庆抱纸取牌
    2. 孙庆递给陈录
    3. 陈录接牌后留下浅脚印方向
  - 禁止越界：
    1. 不定孙庆先取牌
    2. 不定陈录进门
    3. 不证明洗字人
  - 结束状态：两种交牌方向都不能排除
  - 下章承接：补名会中阻止指定单人背锅
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 孙庆抱纸靠近牌匣，负责遮挡
    2. 递牌给陈录，负责反向模拟
    3. 陈录退走浅脚印，负责边界
  - 字数支撑理由：反向动作、遮挡角度和痕迹差异能支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：交牌动作不是推测，而是反向模拟
  - 最低可撑体量判断：能。

## 第209章：都不能排除
- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：曹书吏压补名，王兆陪用两种模拟都不能排除阻止单人背锅
- **本章兑现点**：午后补名压力继续推动会场对抗
- **最晚冲突落点**：前20%
- **核心事件**：曹书吏试图把交牌责任压到孙庆一人身上，王兆陪指出两种模拟都能解释部分痕迹，不能用一个人背锅来补王某全名
- **冲突类型**：会场对抗 / 半证反压
- **人物发展**：主角从查交牌动作转为防止对手制造替罪人
- **信息释放**：对手想让孙庆或陈录单独吸收风险
- **钩子**：赵平说昨夜进门的人拿牌后，门外还剩一个人没走
- **本章不可删场景**：补名会、两种模拟反压、门外剩人必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：两种交牌方向都不能排除
  - 必做事件：
    1. 曹书吏推动补名并压单人责任
    2. 主角用两种模拟反压
    3. 赵平补出门外剩人
  - 禁止越界：
    1. 不定交牌方向
    2. 不证明洗字人
    3. 不撤补名压力
  - 结束状态：交牌线转向谁进门、谁留门外
  - 下章承接：整理交牌半证，转向进门与留门外
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 补名会，负责压力
    2. 两种模拟反压，负责对抗
    3. 门外剩人，负责钩子
  - 字数支撑理由：会场压力、半证边界和新证词能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：交牌模拟进入会场后果
  - 最低可撑体量判断：能。

## 第210章：谁留在门外
- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：交牌动作只能证明夜值牌转过手，下一步必须查谁进洗笔房、谁留在门外
- **本章兑现点**：交牌线阶段收束并转向进门/留门外
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪整理两种交牌模拟、墙缝纸灰、门槛拖痕和赵平“门外剩人”证词，确认夜值牌可能转过手，但仍不能证明最终持牌进门者；下一批必须查门外留人和洗笔房内人
- **冲突类型**：证据边界 / 新方向转移
- **人物发展**：主角继续接受半证推进
- **信息释放**：真正关键不再是谁递牌，而是谁拿牌进洗笔房、谁在门外望风或遮掩
- **钩子**：门外剩人的脚印没有进洗笔房方向，反而朝封袋处退了一步
- **本章不可删场景**：交牌半证整理、不能定进门人、封袋处退步脚印必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：两种交牌模拟都不能排除，赵平说门外剩人
  - 必做事件：
    1. 整理交牌半证
    2. 明确只能证明夜值牌转手
    3. 发现门外剩人退向封袋处
  - 禁止越界：
    1. 不证明最终进门人
    2. 不证明洗字人
    3. 不撤王某疑名
  - 结束状态：下一批转向谁进洗笔房、谁退向封袋处
  - 下章承接：查门外剩人脚印和封袋处退路
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 交牌半证整理，负责收束
    2. 证据边界反压，负责压力
    3. 封袋处退步脚印，负责新方向
  - 字数支撑理由：交牌收束、补名反压和新脚印方向能支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约700~850字
    3. 场景3：约600~750字
  - 摘要风险判断：中
  - 流程章落地检查：交牌不作终局推理，转向进门/留门外现场
  - 最低可撑体量判断：能。
