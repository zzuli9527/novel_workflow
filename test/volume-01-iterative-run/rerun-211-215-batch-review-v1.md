# 第211~215章批次评审（v1）

## 一、总体判断

第211~215章通过了“进门 / 留门外分工不能抽象化”的验证。本批没有把分工写成一句“一个望风一个动手”，而是通过门槛内外湿灰、门外脚跟方向、纸捆遮挡、封袋架擦碰、水盆黑灰和封袋架下沿黑灰完成现场验证。

本批关键验证点成立：

1. 分工线进入门槛、脚印、视线和黑灰现场
2. 两种分工方向都被实际模拟
3. 分工没有直接证明洗字人
4. 陈录、孙庆都没有招供
5. 午后补名和府学查籍压力继续存在
6. 线索转向封袋处接应人与洗笔房水盆痕

## 二、单章结果

- 第211章：门槛内外痕迹成立，脚尖湿灰和脚跟朝封袋处把分工拉回现场
- 第212章：陈录进门、孙庆留门外方向可行但不完整
- 第213章：孙庆进门、陈录留门外方向也可行但不完整
- 第214章：两种分工半证进入补名对抗，压力升级到府学查籍
- 第215章：分工线收束，转向水盆黑灰和封袋架下沿黑灰

## 三、本批主要问题

### 1. 分工动作现场化有效

本批把分工拆成了具体现场：

- 进门者脚尖带湿灰
- 留门外者脚跟朝封袋处
- 孙庆纸捆能遮挡赵平视线和封袋处
- 陈录脚尖湿灰不足
- 孙庆门槛湿灰更吻合
- 陈录封袋处退步角度不吻合
- 水盆黑灰与封袋架下沿黑灰形成新方向

这些都能支撑正文现场，不是抽象推理。

### 2. 半证边界稳定

分工只完成了有限证明：

- 昨夜至少存在进门 / 留门外配合
- 陈录、孙庆两种分工都能解释部分痕迹
- 孙庆更像进门者，但不能解释门外退步痕
- 陈录可解释部分遮挡，但不能解释湿灰

它没有证明：

- 谁最终洗字
- 谁是封袋处接应人
- 王某疑名可以解除
- 可以补王兆陪全名

### 3. 下一批要避免“黑灰线抽象化”

水盆黑灰和封袋架黑灰如果继续往下写，容易变成“黑灰同源，所以有人接应”的抽象判断。下一批必须落到：

- 黑灰颗粒来源
- 袖口抹断痕的高度和方向
- 封袋架下沿是否能被正常取袋动作擦到
- 门外剩人是否必须弯腰或伸手
- 黑灰是否能从水盆带到封袋处

## 四、对工作流的修订结论

本批暂不需要修改工作流。

- `03-chapters.md`：流程章现场动作与人物后果约束有效

## 五、下一批约束

如果继续往下写，重点观察：

1. 封袋处接应人与水盆痕必须进入现场验证
2. 黑灰同源不能直接定案
3. 陈录、孙庆仍不能直接招供
4. 府学查籍压力继续存在
5. 王某疑名不能消失
