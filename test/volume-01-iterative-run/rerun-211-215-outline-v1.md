# 新模具复跑细纲 v1：第211~215章

> 用途：验证“进门 / 留门外分工”是否能落到门槛、脚印、视线、水盆、封袋处等现场，而不是写成一句“一个望风一个动手”。

## 批次目标

- 验证进门人与门外剩人的分工必须通过现场痕迹判断
- 至少模拟两种分工方向：陈录进门 / 孙庆留门外，孙庆进门 / 陈录留门外
- 分工线只能证明有人配合，不能证明谁洗字
- 午后补名压力继续存在
- 王某疑名不能消失

## 批次风险点

- 把分工写成抽象判断
- 让模拟直接定人
- 让陈录或孙庆提前招供
- 让封袋处接应人过早暴露

---

## 第211章：门槛内外

- **功能**：查 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪查洗笔房门槛，发现进门者脚尖带湿灰，门外者脚跟朝封袋处
- **本章兑现点**：进门 / 留门外分工进入现场痕迹
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪回到洗笔房门槛，区分内侧湿灰和外侧退步痕，确认一个人进门时脚尖沾湿灰，另一个人没跨门槛，脚跟却朝封袋处偏了一步
- **冲突类型**：现场勘查 / 痕迹边界
- **人物发展**：主角从交牌动作转入分工验证
- **信息释放**：门外剩人不是单纯站着，他曾朝封袋处调整站位
- **钩子**：门槛内侧湿灰里混着洗笔房水盆边的细黑灰
- **本章不可删场景**：门槛内外痕迹、脚尖湿灰、脚跟朝封袋处必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：交牌只证明夜值牌转手，门外剩人退向封袋处
  - 必做事件：
    1. 查洗笔房门槛内侧湿灰
    2. 查门外退步脚跟方向
    3. 将湿灰与水盆边黑灰关联
  - 禁止越界：
    1. 不证明谁进门
    2. 不证明谁洗字
    3. 不撤补名压力
  - 结束状态：分工必须进入站位模拟
  - 下章承接：模拟陈录进门、孙庆留门外
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 门槛内侧湿灰勘查，负责进门痕迹
    2. 门外脚跟退步勘查，负责留门外痕迹
    3. 水盆黑灰关联，负责下章模拟
  - 字数支撑理由：痕迹区分、衙役干扰、补名催促能支撑展开
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约650~800字
    3. 场景3：约650~850字
  - 摘要风险判断：低
  - 流程章落地检查：分工落到门槛内外痕迹
  - 最低可撑体量判断：能。

## 第212章：陈录进门

- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：模拟陈录进门、孙庆留门外，孙庆纸捆能遮挡封袋处视线，但陈录门槛湿灰不吻合
- **本章兑现点**：第一种分工可行但不完整
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪让陈录按浅脚印跨进洗笔房，让孙庆抱纸留门外。孙庆站位能遮住赵平视线并挡住封袋处，但陈录脚尖沾不到足够湿灰，进门痕迹不吻合
- **冲突类型**：站位模拟 / 半证反压
- **人物发展**：主角确认不能把陈录单定为进门者
- **信息释放**：孙庆留门外可解释遮挡，却不能解释全部脚印
- **钩子**：孙庆抱纸退向封袋处时，纸捆边缘扫到封袋架
- **本章不可删场景**：陈录跨门槛、孙庆纸捆遮挡、湿灰不吻合必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：门槛湿灰和门外退步痕都需要模拟
  - 必做事件：
    1. 陈录按浅脚印进洗笔房
    2. 孙庆抱纸留门外遮挡
    3. 对比陈录脚尖湿灰不足
  - 禁止越界：
    1. 不定陈录进门
    2. 不定孙庆只是望风
    3. 不让孙庆招供
  - 结束状态：陈录进门方向可行但不完整
  - 下章承接：模拟孙庆进门、陈录留门外
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 陈录进门模拟，负责进门动作
    2. 孙庆门外遮挡，负责留门外动作
    3. 湿灰比对，负责证据边界
  - 字数支撑理由：两人站位、视线遮挡、湿灰比对和自保反应能支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约750~900字
    3. 场景3：约550~750字
  - 摘要风险判断：低
  - 流程章落地检查：分工进入站位和痕迹验证
  - 最低可撑体量判断：能。

## 第213章：孙庆进门

- **功能**：查 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：模拟孙庆进门、陈录留门外，门槛湿灰更吻合，但封袋处退步痕不吻合
- **本章兑现点**：第二种分工也只能成立一半
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪让孙庆放下纸捆跨进洗笔房，湿灰和脚尖方向更像昨夜痕迹；陈录留门外退向封袋处时，脚跟角度却偏了半寸，无法完全解释退步痕
- **冲突类型**：反向模拟 / 证据边界
- **人物发展**：主角把两种分工都控制在半证状态
- **信息释放**：孙庆更像进门者，但门外剩人的脚跟角度仍不吻合
- **钩子**：陈录退步不吻合，说明门外剩人可能还接过封袋处某物
- **本章不可删场景**：孙庆跨门槛、陈录退向封袋处、退步痕不吻合必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：陈录进门方向不完整
  - 必做事件：
    1. 孙庆进洗笔房模拟
    2. 陈录留门外退向封袋处模拟
    3. 比对封袋处退步痕角度
  - 禁止越界：
    1. 不定孙庆进门
    2. 不定陈录接应
    3. 不证明封袋处接应人身份
  - 结束状态：两种分工都不能排除，也都不能定案
  - 下章承接：曹书吏借半证继续压补名
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 孙庆进门，负责湿灰吻合
    2. 陈录留门外，负责退步方向
    3. 封袋处角度不吻合，负责新疑点
  - 字数支撑理由：反向模拟、证据边界和两人心理反应能支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约700~850字
    3. 场景3：约650~800字
  - 摘要风险判断：低
  - 流程章落地检查：反向分工仍以现场痕迹判断
  - 最低可撑体量判断：能。

## 第214章：不能补名

- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：曹书吏催午后补名，王兆陪用两种分工都不完整阻止补名
- **本章兑现点**：分工半证进入人物后果和会场对抗
- **最晚冲突落点**：前20%
- **核心事件**：曹书吏抓住“孙庆进门更吻合”逼王兆陪承认王某疑名，王兆陪指出陈录进门能解释遮挡、孙庆进门能解释湿灰，但二者都不能解释封袋处退步痕，因此不能拿半证补名
- **冲突类型**：会场对抗 / 时间压力
- **人物发展**：主角用证据边界拖住补名，不让对手制造定案假象
- **信息释放**：对手急于把“有人分工”写成“王某书生配合洗字”
- **钩子**：曹书吏提出午后前若补不了名，就把疑名移交府学查籍
- **本章不可删场景**：午后补名压力、两种分工反压、府学查籍威胁必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：两种分工都只有半证
  - 必做事件：
    1. 曹书吏压午后补名
    2. 主角用分工半证反压
    3. 府学查籍威胁出现
  - 禁止越界：
    1. 不让补名压力消失
    2. 不让王某疑名解除
    3. 不证明洗字人
  - 结束状态：补名被拖住但压力升级
  - 下章承接：分工线转向封袋处接应人和水盆痕
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 补名会场施压，负责冲突
    2. 两种分工对照，负责反压
    3. 府学查籍威胁，负责后果
  - 字数支撑理由：会场对抗、证据边界和新威胁能支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：分工半证产生人物后果
  - 最低可撑体量判断：能。

## 第215章：水盆黑灰

- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：分工线只能证明有人配合，真正新方向是封袋处接应人和洗笔房水盆痕
- **本章兑现点**：分工线阶段收束，转入下一轮实跑方向
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪复盘门槛湿灰、门外退步、纸捆遮挡和两种分工，确认有人配合但不能定洗字人；他重新查水盆边黑灰，发现黑灰不只在门槛，也沾在封袋架下沿
- **冲突类型**：证据边界 / 新方向转移
- **人物发展**：主角接受分工半证，继续向接应链推进
- **信息释放**：洗笔房内水盆痕与封袋处可能存在同一黑灰来源
- **钩子**：封袋架下沿黑灰有一道被袖口抹断的痕
- **本章不可删场景**：分工半证收束、水盆黑灰、封袋架下沿黑灰必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：两种分工都无法定案，午后补名压力升级
  - 必做事件：
    1. 复盘分工证据边界
    2. 查水盆边黑灰
    3. 查封袋架下沿黑灰
  - 禁止越界：
    1. 不证明接应人身份
    2. 不证明谁洗字
    3. 不撤王某疑名
  - 结束状态：下一批转向封袋处接应人与水盆痕
  - 下章承接：查封袋架下沿黑灰来源
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 分工半证收束，负责边界
    2. 水盆黑灰复查，负责新证据
    3. 封袋架下沿黑灰，负责钩子
  - 字数支撑理由：半证复盘、黑灰比对和补名压力能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约700~850字
    3. 场景3：约650~850字
  - 摘要风险判断：中
  - 流程章落地检查：分工不作终局推理，转向可验证的新现场
  - 最低可撑体量判断：能。
