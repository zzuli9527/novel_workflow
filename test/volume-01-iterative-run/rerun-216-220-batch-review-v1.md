# 第216~220章批次评审（v1）

## 一、总体判断

第216~220章通过了“黑灰线不能抽象化”的验证。本批没有把水盆黑灰和封袋架黑灰写成一句“黑灰同源，所以有人接应”，而是通过三份灰样比对、细砂感、正常取袋排除、弯腰取架底物、扶架伸手脚跟角度、短促木响和封袋编号错位完成现场验证。

本批关键验证点成立：

1. 黑灰线进入颗粒、袖口、架底高度和脚跟角度
2. 正常取袋与弯腰取架底物完成动作区分
3. 黑灰没有直接证明接应人身份
4. 陈录、孙庆都没有招供
5. 府学查籍和王某疑名压力继续存在
6. 线索转向架底被藏物和封袋编号错位

## 二、单章结果

- 第216章：三份黑灰比对成立，细砂感成为物证差异，但不能定同源
- 第217章：正常取袋碰不到下沿黑灰，弯腰取架底物才能擦断黑灰
- 第218章：扶架伸手能解释脚跟角度，但仍不能定人
- 第219章：黑灰半证进入府学查籍压力，赵平补出架下木响
- 第220章：黑灰线收束为架底取放动作，转向编号错位

## 三、本批主要问题

### 1. 黑灰动作现场化有效

本批把黑灰拆成了具体现场：

- 水盆黑灰、封袋架黑灰和普通墨灰三份比对
- 黑灰细砂感不等于同源定案
- 正常取封袋碰不到下沿黑灰
- 弯腰取架底物会擦断黑灰
- 扶架伸手时脚跟角度与退步痕吻合
- 封袋架下短促木响
- 编号错位与架底磨痕相连

这些都能支撑正文现场，而不是抽象推理。

### 2. 半证边界稳定

黑灰只完成了有限证明：

- 水盆和封袋架下沿可能存在接触链
- 封袋架下沿不是正常取袋动作留下
- 门外剩人可能扶架弯腰取放过架底物
- 架底可能藏过薄木片、封条或临时垫位

它没有证明：

- 黑灰一定同源
- 谁取放架底物
- 谁是封袋处接应人
- 王某疑名可以解除
- 可以补王兆陪全名

### 3. 下一批要避免“编号线抽象化”

封袋编号错位如果继续往下写，容易变成“编号错了，所以封袋被换”的抽象判断。下一批必须落到：

- 错位前后的封袋记录
- 封袋架格位宽度
- 薄木片 / 封条 / 临时垫位的具体尺寸
- 王某疑名封袋为何被避开
- 编号错位是旧错还是昨夜新错
