# 新模具复跑细纲 v1：第216~220章

> 用途：验证“水盆黑灰 / 封袋架黑灰”是否进入颗粒、袖口、架底高度、取袋动作和脚跟角度的现场验证，而不是写成“黑灰同源，所以有人接应”。

## 批次目标

- 验证黑灰线必须落到颗粒、触碰高度、取放动作和脚跟方向
- 至少模拟一次正常取封袋与弯腰取架底物的差异
- 黑灰只能证明水盆和封袋处可能有关联，不能证明接应人身份
- 府学查籍压力继续存在
- 王某疑名不能消失

## 批次风险点

- 黑灰比对后直接定案
- 接应人过早暴露
- 府学查籍压力被写丢
- 陈录、孙庆被迫直接招供

---

## 第216章：黑灰带砂

- **功能**：查 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪比对水盆、封袋架和普通墨灰，发现两处黑灰都有细砂感
- **本章兑现点**：黑灰线进入颗粒比对，但不能定同源
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪将水盆边黑灰、封袋架下沿黑灰和书吏案头墨灰分别碾开，发现前两者都有细砂，普通墨灰没有；但细砂只能说明可能来自洗笔房水盆或潮湿地面，不能证明谁带过去
- **冲突类型**：物证比对 / 证据边界
- **人物发展**：主角从分工线转向物证接触链
- **信息释放**：黑灰不是普通墨灰，可能与水盆边沉灰有关
- **钩子**：赵平说洗笔房水盆昨夜曾被人往墙根挪过
- **本章不可删场景**：三份黑灰比对、细砂感、不能定人必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：分工线只能证明有人配合，水盆黑灰与封袋架黑灰形成新方向
  - 必做事件：
    1. 比对水盆黑灰
    2. 比对封袋架下沿黑灰
    3. 比对普通墨灰并给出边界
  - 禁止越界：
    1. 不证明黑灰同源
    2. 不证明接应人身份
    3. 不撤王某疑名
  - 结束状态：黑灰需要进入取放动作模拟
  - 下章承接：模拟正常取封袋是否会擦到下沿
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 三份黑灰取样，负责物证入口
    2. 碾灰比对，负责差异
    3. 赵平补水盆移位，负责钩子
  - 字数支撑理由：物证操作、曹书吏质疑和赵平回忆能支撑展开
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约500~700字
  - 摘要风险判断：低
  - 流程章落地检查：黑灰落到取样和颗粒触感
  - 最低可撑体量判断：能。

## 第217章：正常取袋

- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：模拟正常取封袋，手和袖口碰不到架下沿黑灰，除非弯腰从架底取物
- **本章兑现点**：封袋架黑灰进入取袋动作验证
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪让赵平、陈录、孙庆分别按正常取封袋动作伸手，袖口都碰不到下沿黑灰；只有弯腰伸手从架底摸东西时，袖口才会擦断黑灰
- **冲突类型**：动作模拟 / 排除正常动作
- **人物发展**：主角把黑灰从“相似”推进到“触碰姿势”
- **信息释放**：封袋架下沿黑灰不是正常取袋留下
- **钩子**：架底空隙里有一道新磨痕，像曾塞过薄木片或小封条
- **本章不可删场景**：正常取袋模拟、弯腰取架底物、袖口擦断黑灰必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：两处黑灰都有细砂感，但不能定同源
  - 必做事件：
    1. 模拟正常取封袋
    2. 模拟弯腰取架底物
    3. 查出架底新磨痕
  - 禁止越界：
    1. 不证明谁取过架底物
    2. 不证明接应人身份
    3. 不让陈录、孙庆招供
  - 结束状态：黑灰指向架底取放动作
  - 下章承接：模拟门外剩人退向封袋处的脚跟角度
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 正常取袋，负责排除
    2. 弯腰取物，负责触碰动作
    3. 架底磨痕，负责新线索
  - 字数支撑理由：多次模拟、人物自保和封袋架细节能支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：黑灰不靠判断定案，而靠动作排除
  - 最低可撑体量判断：能。

## 第218章：扶架伸手

- **功能**：查 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：模拟门外剩人退向封袋处，只有左手扶架、右手伸下沿时，脚跟角度与退步痕吻合
- **本章兑现点**：退步痕与封袋架下沿触碰动作被连上
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪让陈录、孙庆按门外剩人退步痕模拟。普通后退角度不对，侧身让路也不对；只有左手扶封袋架、右手伸向架底时，脚跟会斜压出相同角度
- **冲突类型**：站位模拟 / 半证推进
- **人物发展**：主角确认门外剩人可能做过取放动作，但仍不能定是谁
- **信息释放**：封袋处不是单纯接应人站位，而是可能有架底取放动作
- **钩子**：曹书吏抢先说“这只证明有人摸架底”，逼王兆陪承认证据边界
- **本章不可删场景**：普通后退排除、扶架伸手模拟、脚跟角度吻合必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：封袋架下沿黑灰指向弯腰取架底物
  - 必做事件：
    1. 模拟普通后退
    2. 模拟侧身让路
    3. 模拟扶架伸手并吻合脚跟痕
  - 禁止越界：
    1. 不证明谁扶架伸手
    2. 不证明架底物是什么
    3. 不撤府学查籍压力
  - 结束状态：架底取放动作成立为可能
  - 下章承接：曹书吏用证据边界压府学查籍
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 普通后退排除，负责边界
    2. 扶架伸手模拟，负责新动作
    3. 曹书吏反压，负责后果
  - 字数支撑理由：动作复现、痕迹角度和会场压力能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：接应线落到脚跟角度和扶架伸手
  - 最低可撑体量判断：能。

## 第219章：只能证明接触

- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：曹书吏压府学查籍，王兆陪用“黑灰只能证明接触动作，不能证明人名”阻止补名
- **本章兑现点**：黑灰半证进入程序压力
- **最晚冲突落点**：前20%
- **核心事件**：曹书吏抓住王兆陪承认证据边界，要求午后前移交府学查籍；王兆陪承认黑灰只能证明有人接触架底，却反过来指出不能用“有人接触”补成“王兆陪接触”
- **冲突类型**：会场对抗 / 程序威胁
- **人物发展**：主角继续用半证边界阻止假案落册
- **信息释放**：对手想把黑灰线转成程序压力，而不是继续查物证
- **钩子**：赵平补出昨夜封袋架下曾有短促木响
- **本章不可删场景**：府学查籍施压、黑灰证据边界、木响线索必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：扶架伸手动作可能成立，但不能定人
  - 必做事件：
    1. 曹书吏压府学查籍
    2. 主角承认证据边界并反压补名
    3. 赵平补出架下木响
  - 禁止越界：
    1. 不解除王某疑名
    2. 不证明架底物身份
    3. 不让黑灰直接定案
  - 结束状态：查籍压力继续，木响引向架底被藏物
  - 下章承接：查架底磨痕与封袋编号错位
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 查籍压力，负责外部后果
    2. 证据边界反压，负责冲突
    3. 木响补证，负责钩子
  - 字数支撑理由：程序压力、主角身份风险和新听觉线索能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：物证半证进入人物和程序后果
  - 最低可撑体量判断：能。

## 第220章：编号错位

- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：黑灰线只能证明架底有人取放过东西，真正新方向是架底被藏物和封袋编号错位
- **本章兑现点**：黑灰线阶段收束，转向下一轮可验证现场
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪查架底磨痕，确认它更像薄木片或封条被抽出；封袋架上编号顺序有一个空位错开半格，说明有人动过架底物或临时替换过封袋位置
- **冲突类型**：证据边界 / 新方向转移
- **人物发展**：主角接受黑灰只能证明接触动作，继续向藏物和编号错位推进
- **信息释放**：架底可能藏过薄木片、封条或临时封袋；编号错位会引向更具体的封袋链
- **钩子**：错位编号正好避开王某疑名所在的那份封袋
- **本章不可删场景**：黑灰半证收束、架底磨痕、封袋编号错位必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：黑灰证明接触动作，赵平补出架下木响
  - 必做事件：
    1. 查架底磨痕
    2. 查封袋编号顺序
    3. 发现错位编号避开王某疑名封袋
  - 禁止越界：
    1. 不证明藏物内容
    2. 不证明接应人身份
    3. 不撤查籍压力
  - 结束状态：下一批转向架底被藏物和封袋编号错位
  - 下章承接：查编号错位前后的封袋记录
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 架底磨痕，负责承接黑灰
    2. 编号顺序，负责新线索
    3. 王某封袋被避开，负责钩子
  - 字数支撑理由：黑灰收束、编号比对和补名压力能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~900字
    3. 场景3：约650~800字
  - 摘要风险判断：中
  - 流程章落地检查：黑灰不作终局推理，转向编号和藏物现场
  - 最低可撑体量判断：能。
