# 第221~225章批次评审（v1）

## 一、总体判断

第221~225章通过了“编号线不能抽象化”的验证。本批没有把封袋编号错位写成一句“编号错位，所以封袋被换”，而是通过封袋格位测量、半格宽度、薄木片抽出模拟、旧点验记录、旧错 / 新动边界、字迹停顿和临时封条尺寸完成现场验证。

本批关键验证点成立：

1. 编号错位进入格位、尺寸、记录和动作模拟
2. 抽出薄木片可复现半格错位，但没有证明封袋被换
3. 王某疑名封袋没有被过早洗清
4. 陈录、孙庆都没有直接招供
5. 府学查籍压力继续存在
6. 线索转向急件房 / 府学来文的临时封条领用记录

## 三、本批主要问题

### 1. 编号错位现场化有效

本批把编号错位拆成了具体现场：

- 封袋格位宽度
- 错位半格宽度
- 架底旧压痕
- 薄木片垫入和抽出
- 抽出木片产生短促木响
- 三日前旧点验小注
- 小注字迹停顿
- 临时封条尺寸吻合半格

这些都能支撑正文现场，不是抽象判断。

### 2. 半证边界稳定

编号线只完成了有限证明：

- 封袋架底可能被垫过或抽过东西
- 抽出薄木片可造成半格错位
- 三日前已有旧错
- 昨夜可能有人借旧错新动
- 临时封条比薄木片更贴近半格宽度和木响

它没有证明：

- 封袋被换
- 王某疑名封袋被动过
- 瘦书办作案
- 府学参与
- 王某疑名可以解除

### 3. 下一批要避免“封条来源抽象化”

临时封条来源如果继续往下写，容易变成“尺寸吻合，所以来自急件房 / 府学”的抽象判断。下一批必须落到：

- 急件房临时封条领用簿
- 府学来文封条样式
- 领用人、交接人和空白封条数量
- 封条卷曲后能否塞入架底
- 封条上是否会沾水盆黑灰或架底木屑
