# 新模具复跑细纲 v1：第221~225章

> 用途：验证“封袋编号错位”是否进入格位、木片宽度、封条尺寸、旧记录和移动痕的现场验证，而不是写成“编号错位，所以封袋被换”。

## 批次目标

- 验证编号错位必须落到封袋格位、半格宽度、木片 / 封条尺寸和旧记录
- 至少模拟一次抽出薄木片或临时封条后，封袋是否会错半格
- 编号错位只能证明封袋架曾被动过，不能证明王某封袋被换
- 府学查籍压力继续存在
- 王某疑名不能消失

## 批次风险点

- 编号错位直接定案
- 王某疑名封袋被过早洗清
- 薄木片 / 封条变成空泛物件
- 府学查籍压力被写丢

---

## 第221章：半格宽度

- **功能**：查 / 得
- **目标字数**：2000~2500字
- **本章即时卖点**：王兆陪量封袋格位，确认错半格不是手滑，必须有薄木片或临时封条垫过
- **本章兑现点**：编号错位进入格位和尺寸现场
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪用竹签量封袋格位、错位半格和架底磨痕，发现错出的半格宽度接近一条临时封条或薄木片；但这只能说明架底曾被垫过或抽过，不能证明封袋被换
- **冲突类型**：现场测量 / 证据边界
- **人物发展**：主角从黑灰线转入编号结构验证
- **信息释放**：错半格需要具体垫物，不是随手摆歪
- **钩子**：旧封袋架边缘有一道同宽压痕
- **本章不可删场景**：量格位、量半格、比对薄木片 / 封条宽度必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：黑灰线收束为架底取放动作，封袋编号半格错位
  - 必做事件：
    1. 测量封袋格位宽度
    2. 测量错位半格宽度
    3. 将半格宽度与薄木片 / 封条关联
  - 禁止越界：
    1. 不证明封袋被换
    2. 不证明王某封袋安全
    3. 不撤府学查籍压力
  - 结束状态：编号错位需要进入抽垫物模拟
  - 下章承接：模拟抽出薄木片后的错位
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 封袋格位测量，负责结构入口
    2. 半格宽度比对，负责错位原因
    3. 旧压痕发现，负责钩子
  - 字数支撑理由：测量动作、曹书吏质疑和尺寸比对能支撑展开
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约750~900字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：编号错位落到尺寸和压痕
  - 最低可撑体量判断：能。

## 第222章：抽出木片

- **功能**：查 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：模拟抽出薄木片，封袋会顺势错半格，但王某疑名封袋不会被直接移动
- **本章兑现点**：编号错位进入可复现动作
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪让赵平用同宽薄木片垫入架底，再抽出。底层封袋会往前滑半格，错位形状吻合；但王某疑名封袋只是被旁边位置牵动，并没有被直接取出或替换
- **冲突类型**：动作模拟 / 半证反压
- **人物发展**：主角确认错位可由抽垫物造成，但拒绝直接推出换袋
- **信息释放**：有人可借错位藏取物，同时避开直接动王某封袋
- **钩子**：抽木片时会发出与赵平所说相似的短促木响
- **本章不可删场景**：垫木片、抽木片、王某封袋未被直接移动必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：半格宽度指向薄木片 / 临时封条
  - 必做事件：
    1. 垫入同宽薄木片
    2. 抽出薄木片并复现半格错位
    3. 确认王某封袋未被直接抽换
  - 禁止越界：
    1. 不证明谁抽木片
    2. 不证明封袋被换
    3. 不解除王某疑名
  - 结束状态：编号错位可复现，但仍是半证
  - 下章承接：查旧封袋记录判断旧错还是新动
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 垫木片，负责实验条件
    2. 抽木片，负责复现错位
    3. 王某封袋边界，负责半证
  - 字数支撑理由：动作复现、木响和会场反应能支撑
  - 场景级字数预算：
    1. 场景1：约700~850字
    2. 场景2：约800~950字
    3. 场景3：约550~700字
  - 摘要风险判断：低
  - 流程章落地检查：编号错位不是判断，而是模拟复现
  - 最低可撑体量判断：能。

## 第223章：旧错新动

- **功能**：查 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：查旧封袋记录，发现错位可能早已有之，也可能昨夜被借用
- **本章兑现点**：编号错位进入记录对照，避免直接定案
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪查封袋架旧点验记录，发现三日前已有一次“底层右二半格偏”的小注；但昨夜黑灰、木响和新磨痕又证明这处旧错可能被再次利用
- **冲突类型**：记录对照 / 新旧边界
- **人物发展**：主角确认不能把旧错直接写成昨夜换袋，但也不能把它当无关旧病
- **信息释放**：对手可能借旧错藏新动
- **钩子**：三日前点验记录的签名是瘦书办代填
- **本章不可删场景**：旧点验记录、三日前小注、新磨痕对照必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：抽木片可复现半格错位，但不能定昨夜换袋
  - 必做事件：
    1. 查旧点验记录
    2. 找到三日前半格偏小注
    3. 对照昨夜新磨痕和黑灰
  - 禁止越界：
    1. 不证明三日前就是布局
    2. 不证明昨夜一定换袋
    3. 不让瘦书办直接暴露
  - 结束状态：旧错 / 新动未分
  - 下章承接：曹书吏借未分边界压府学查籍
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 旧记录查找，负责记录入口
    2. 三日前小注，负责旧错
    3. 新磨痕对照，负责新动可能
  - 字数支撑理由：记录追查、人物代填和证据边界能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约750~900字
    3. 场景3：约650~800字
  - 摘要风险判断：低
  - 流程章落地检查：编号错位进入记录对照
  - 最低可撑体量判断：能。

## 第224章：旧错未分

- **功能**：压 / 争
- **目标字数**：2000~2500字
- **本章即时卖点**：曹书吏压查籍，王兆陪用“旧错 / 新动未分”阻止把错位写成换袋
- **本章兑现点**：编号半证进入程序对抗
- **最晚冲突落点**：前20%
- **核心事件**：曹书吏抓住三日前已有错位，要求把昨夜线索归为旧错并启动府学查籍；王兆陪指出旧错不能证明昨夜无事，新磨痕也不能证明封袋被换，因此两边都不能写死
- **冲突类型**：会场对抗 / 程序威胁
- **人物发展**：主角继续用边界阻止对手把半证写成定论
- **信息释放**：对手想借旧错消解新动，再推进查籍
- **钩子**：瘦书办否认代填，但字迹停顿与旧小注不一致
- **本章不可删场景**：查籍施压、旧错 / 新动边界、瘦书办字迹停顿必须落地
- **视角**：王兆陪
- **节奏**：双峰
- **写作交接卡**：
  - 开场状态：三日前小注与昨夜新磨痕并存
  - 必做事件：
    1. 曹书吏压府学查籍
    2. 主角用旧错 / 新动未分反压
    3. 瘦书办字迹停顿露出新方向
  - 禁止越界：
    1. 不证明瘦书办作案
    2. 不解除王某疑名
    3. 不证明封袋被换
  - 结束状态：编号线继续，但转向薄木片来源和封条尺寸
  - 下章承接：收束编号线，确定下一批查物件来源
- **章节可写性证明**：
  - 场景数：3
  - 场景拆分：
    1. 查籍施压，负责外部压力
    2. 旧错 / 新动边界，负责对抗
    3. 字迹停顿，负责钩子
  - 字数支撑理由：程序压力、证据边界和人物反应能支撑
  - 场景级字数预算：
    1. 场景1：约650~800字
    2. 场景2：约850~1000字
    3. 场景3：约500~650字
  - 摘要风险判断：低
  - 流程章落地检查：编号半证产生程序后果
  - 最低可撑体量判断：能。

## 第225章：封条尺寸

- **功能**：得 / 压
- **目标字数**：2000~2500字
- **本章即时卖点**：编号线只能证明有人借错位藏取物，下一步转向薄木片来源和封条尺寸
- **本章兑现点**：编号错位阶段收束，转入下一轮物件来源验证
- **最晚冲突落点**：前25%
- **核心事件**：王兆陪复盘格位宽度、抽木片模拟、旧记录和新磨痕，确认编号线不能证明换袋，只能证明有人借错位藏取物；他比对衙门常用封条尺寸，发现有一种临时封条宽度正好等于半格

- **人物发展**：主角接受编号半证，继续向具体物件来源推进
- **信息释放**：临时封条比薄木片更符合半格宽度和短促木响
- **钩子**：临时封条只在急件房和府学来文中常用
- **本章不可删场景**：编号线收束、封条尺寸比对、急件房 / 府学来文方向必须落地
- **视角**：王兆陪

    1. 收束编号错位证据边界
    2. 比对常用封条尺寸
    3. 指向急件房和府学来文
  - 禁止越界：
    1. 不证明封条来源
    2. 不证明府学参与
    3. 不撤王某疑名
  - 结束状态：下一批转向薄木片来源和封条尺寸
  - 下章承接：查急件房临时封条领用记录
- **章节可写性证明**：

  - 场景拆分：

    2. 封条尺寸比对，负责新线索
    3. 急件房 / 府学来文方向，负责钩子
  - 字数支撑理由：半证复盘、尺寸比对和新机构方向能支撑

    2. 场景2：约750~900字
    3. 场景3：约650~800字
