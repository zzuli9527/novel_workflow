## 任务

## 核心概念：交叉模式

多条线交汇时，有几种基本交叉模式：

### 模式1：乒乓（交替推进）
```
线A → 线B → 线A → 线B → 汇合
```
适用于：两条线势均力敌，互相影响。读者在两条线之间来回切换。

### 模式2：螺旋（逐步靠近）
```
线A ──┐
       ├── 线A&B 接触 ──┐
线B ──┘                 ├── 汇合
线C ────────────────────┘
```
适用于：多条线从独立推进到逐渐接触再到完全汇合。模块A（多线并拢型）的标准展开方式。

### 模式3：引爆（一条线打破另一条）
```
线A ────────────┬── 线A被冲击
                │
线B ── 引爆点 ──┘
```
适用于：一条线的事件突然闯入另一条线，打破原有节奏。常用于"转"节拍。

### 模式4：同步（平行剪辑）
```
线A：事件A1 → 事件A2 → 事件A3
线B：事件B1 → 事件B2 → 事件B3
（最后汇合）
```
适用于：多条线在同步推进但尚未接触时，通过平行剪辑制造"命运交汇"的预感。
